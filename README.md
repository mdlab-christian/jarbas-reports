## Estrutura

```
jarbas-reports/
├── design-system/     → DS v7, tokens, componentes, propostas
├── lapidacoes/        → Relatórios de lapidação (fase 1 investigação)
├── olivia/            → OlivIA 3.0 — planos, PRDs, protótipos
├── controller/        → Controller V2 — planos, UI, relatórios
├── processos/         → Página /processos
├── financeiro/        → Página /financeiro
├── acordos/           → Página /acordos
├── decisoes/          → Página /decisoes
├── execucoes/         → Página /execucoes
├── audiencias/        → Página /audiencias
├── distribuicao/      → Página /distribuicao
├── jurimetria/        → Página /jurimetria
├── pesquisas/         → Deep research, investigações, auditorias
└── misc/              → Tudo que não se encaixa nas categorias acima
```
