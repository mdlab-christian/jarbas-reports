nome TEXT NOT NULL

area_atuacao TEXT

💡 Lógica Condicional

Se trabalha_direito = true → mostrar 3 campos adicionais com animação suave.

Se trabalha_direito = false → só coletar contato e agradecer. Pode segmentar como "parceiro potencial".

🎯 Segmentação automática

Capturar dados → salvar em Supabase (tabela waitlist)
n8n webhook → notificar Christian no Telegram
Score automático: >1000 processos → lead MDLab Enterprise
Email automático de confirmação (n8n + Resend)

Sugestão copy botão: "Quero Entrar na Lista →" funciona melhor que "Cadastrar" ou "Enviar" — mais ativo, mais exclusivo.
