Multi-Grupo Telegram — Como Configurar

Suporte a múltiplos grupos implementado. Para ativar:

  1. Adicionar ao ~/jarbas/.env:
    TELEGRAM_EXTRA_GROUPS=chatId1,chatId2,chatId3
  2. Cada grupo pode ter um contexto de página (ex: grupo de /olivia, grupo de /controller)
  3. Reiniciar JARBAS após adicionar os IDs
  4. Próximo passo P1: adicionar comando /set_context /PAGINA para mapear grupo → lapidação
