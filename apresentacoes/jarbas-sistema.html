O que é o JARBAS

Sistema de browser automation jurídica que roda 24/7 num Mac Mini M4. Captura intimações do EPROC, analisa documentos com IA, distribui processos e orquestra o ecossistema MdFlow — tudo de forma autônoma via fila de tarefas no Supabase.

Como funciona

O Christian (ou os agentes IA) envia comandos via Telegram. O JARBAS enfileira tarefas, executa skills especializadas com Playwright + Claude, e notifica o resultado. Funciona nos schedules 06h, 13h e 19h automaticamente.

Stack técnico

Node.js ESM · Playwright (Chromium) · Anthropic Claude (Sonnet 4.6 + Haiku 4.5) · Supabase (PostgreSQL + Storage) · Telegram Bot API · ElevenLabs TTS · OpenClaw Gateway.
