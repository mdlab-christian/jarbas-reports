FRONT-D05 Hook/Query CAT-B Design System Esforço Baixo

DrawerTabChecklist — sem staleTime (padrão 0 = refetch sempre)

DrawerTabChecklist.tsx:33 — O hook useQuery do checklist não define staleTime. Segundo o CLAUDE.md, "staleTime: 30_000 em TODOS os hooks React Query". Sem staleTime, toda re-montagem do componente dispara um refetch desnecessário para o banco.

ANTES — staleTime ausente
const { data } = useQuery({
  queryKey: ['checklist', id],
  queryFn: async () => {...},
  enabled: !!id && !!org,
  // staleTime: ausente → 0ms
});
DEPOIS — staleTime: 30_000
const { data } = useQuery({
  queryKey: ['checklist', id],
  queryFn: async () => {...},
  enabled: !!id && !!org,
  staleTime: 30_000,
});

Arquivo: src/pages/atividades/drawer/DrawerTabChecklist.tsx:46 — Adicionar staleTime: 30_000 após enabled.
