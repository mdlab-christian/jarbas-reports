5. Achados Detalhados

P1 BUG-001: ModalCategoria "Editar" button sem acao
No modo view com perfil admin, o botao "Editar" no footer da ModalCategoria tem onClick={() => {}} (funcao vazia). Clicar nao faz nada. Deveria trocar o modo de view para edit.
src/pages/empresas/modals/ModalCategoria.tsx:250
P1 BUG-002: ModalEscritorio delete sem check de perfil
O botao "Excluir" no ModalEscritorio aparece para qualquer usuario nos modos edit e view, sem verificar se o perfil e admin. Outros modais (ModalOrgao, CardEmpresaGrande) fazem o check correto de perfil === 'admin'.
src/pages/empresas/modals/ModalEscritorio.tsx:545-555
P2 STUB-001: MetricasTab do Escritorio retorna zeros hardcoded
A tab Metricas do ModalEscritorio sempre mostra 0 para processos, acordos, taxa de acordo e valor medio. A queryFn retorna um objeto com zeros hardcoded em vez de buscar dados reais do banco.
src/pages/empresas/modals/ModalEscritorio.tsx (MetricasTab queryFn)
P2 STUB-002: useCategoriasComCount nao filtra por ativa=true
O hook useCategoriasComCount na tab Categorias nao filtra categorias inativas. Categorias inativas aparecem na lista junto com as ativas, sem distincao visual clara. O hook useCategorias (usado no filtro dropdown) JA filtra corretamente por eq('ativa', true).
src/pages/empresas/hooks/useCategorias.ts
P2 UX-001: CNPJ sem mascara no ModalOrgao edit
Ao abrir ModalOrgao em modo edit, o campo CNPJ exibe digitos brutos (ex: "99888777000166") em vez do formato mascarado (99.888.777/0001-66). A mascara so e aplicada ao digitar (onChange), mas nao ao carregar o valor inicial.
src/pages/empresas/modals/ModalOrgao.tsx:59
P2 UX-002: Escritorios tab sem paginacao
A tab Escritorios lista todos os 80+ registros sem paginacao server-side. Para volumes maiores isso pode causar lentidao. Orgaos e Categorias tem o mesmo pattern (listas completas sem paginacao), mas possuem volumes menores.
src/pages/empresas/hooks/useEscritorios.ts

6. Features Especificas

CSV Export

  • Botao "Exportar" visivel apenas na tab Empresas
  • Codigo busca todas empresas filtradas (sem paginacao)
  • Gera BOM UTF-8 para compatibilidade Excel
  • Nao testado end-to-end (download file dialog)

CSV Import

  • Botao "Importar" visivel na tab Empresas
  • Valida coluna "nome_fantasia" obrigatoria
  • Processa em batch com callWebhook
  • Nao testado end-to-end (file upload)

OlivIA Button

  • Botao renderiza no header da pagina
  • Segundo botao OlivIA na barra de acoes
  • Nao testado (requer integracao OlivIA ativa)

Dirty State / Discard

  • ModalEmpresa detecta mudancas via JSON.stringify
  • Dialog "Descartar alteracoes?" ao fechar com mudancas
  • ModalEscritorio dirty state funcional
  • Botoes Cancelar/Confirmar no dialog de discard

Enrichment IA (ReceitaWS)

  • Botao "Enriquecer" no tab Cadastro do ModalEmpresa
  • Chama webhook empresas/enriquecer
  • Nao testado end-to-end (requer CNPJ valido + API)

Permissoes por Perfil

  • Nova Empresa: hidden para operacional
  • Delete empresa: admin-only no CardEmpresaGrande
  • Mesclar: admin + juridico
  • Nova Categoria: admin-only
  • Delete escritorio: sem check de perfil (BUG-002)
