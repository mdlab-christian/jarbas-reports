CSS only risco baixo 📂 3 componentes: daily_agenda, gestor_executive_block, mural_avisos

DS v5 define: "H1, H2 → Manrope (font-display). Labels de seção/card usam Inter por default, mas o DS especifica que títulos de seção dentro de cards devem manter a hierarquia visual com Manrope." Os 3 h3 de seção ("Agenda do Dia", "Comercial/Jurídico", "Mural de Avisos") usam text-sm font-semibold sem font-display. Visualmente imperceptível em font-size pequeno, mas inconsistente com a regra de tipografia para títulos de componente. A PASSO 2 pré-carregado já identificou este achado.
