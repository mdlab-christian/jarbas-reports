Contexto da Exploração SETUP
Dados reais: Org Midas com ~14.456 processos totais, 7.489 ativos, taxa de sucesso 47.7%. Julgadores [DEEP] criados via Supabase REST API (upsert_julgador) e removidos ao final (ativo=false). Navegação via history.pushState + popstate — direto via URL é bloqueado pelo PageGuard durante hidratação.
Limitação de exploração: O token JWT do Supabase expira durante a sessão longa de exploração. Após cada expiração, agent-browser eval redireciona para /homepage (comportamento de proteção do AuthContext). Causou múltiplos re-logins durante a sessão, mas não impactou a qualidade dos achados.
