FRONT-010 — Backend Required: Avatar foto do contato

O avatar de contatos sempre exibe a inicial do nome. Para suportar foto, é necessário:

  • Campo photo_url TEXT na tabela whatsapp_conversas ou na tabela de leads/clientes
  • Integração com Evolution API para sincronizar a foto do perfil WhatsApp
  • Componente Avatar com fallback para inicial quando photo_url for null

Aprovação CTO necessária (adicionar campo em tabela de dados).
