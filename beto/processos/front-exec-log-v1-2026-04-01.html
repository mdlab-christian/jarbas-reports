FRONT-002 — Placeholder corrigido + enforcement mín. 2 chars

Placeholder dizia "mín. 4 caracteres" mas o debounce disparava em qualquer quantidade. Corrigido: mín. 2 chars (padrão UX) + placeholder atualizado.

// debounce sem enforcement:
const timer = setTimeout(() => onUpdateFiltro('search', localSearch), 300);

// placeholder enganoso:
placeholder="Buscar por cliente, empresa ou nº CNJ... (mín. 4 caracteres)"
// enforcement real:
const value = localSearch.trim();
if (value.length > 0 && value.length < 2) return;
const timer = setTimeout(() => onUpdateFiltro('search', localSearch), 300);

// placeholder preciso:
placeholder="Buscar por cliente, empresa, advogado ou nº CNJ..."

FRONT-005 — Breadcrumb tipografia

&gt; é um hack HTML. O símbolo correto para breadcrumb é › (U+203A).

breadcrumb={<span>Home &gt; Processos</span>}
breadcrumb={<span>Home › Processos</span>}
