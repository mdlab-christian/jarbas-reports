04 — Jarbas.OS Section

Como Posicionar o Jarbas.OS

Não é um hardware. É um sistema operacional completo de agentes autônomos. O OpenClaw por baixo — mas o que o usuário vê é o OS.

✅ Como posicionar

  • →Sistema operacional completo de agentes jurídicos autônomos
  • →Cada agente tem especialização: distribuição, monitoramento, análise, acordos
  • →Orquestração central — todos os agentes trabalham coordenados
  • →On-premise: dados nunca saem da infraestrutura do cliente
  • →Escala horizontal: mais agentes conforme a necessidade cresce
  • →Funciona 24/7 sem intervenção humana

❌ O que NÃO mencionar

  • ✗Mac Mini M4 / hardware específico
  • ✗"1 nó ativo" / quantidade de máquinas
  • ✗OpenClaw (tecnologia interna, não revelar)
  • ✗Preços da concorrência (Jusbrasil)
  • ✗"em expansão futura" — falar como pronto

Copy sugerida — Headline + Sub

HEADLINE Jarbas.OS — Seu Escritório Nunca Para

4 modos no mobile — opção recomendada

Swipeable horizontal cards

4 cards horizontais com scroll snap. Indicador de dots embaixo. O usuário swipa para ver cada modo. Funciona naturalmente no touch.

Accordion

Cada modo é um item de accordion. Clica para expandir. Mais simples, menos impactante visualmente.

Recomendação: Swipeable cards no mobile, grid 4 colunas no desktop. Framer Motion com drag para implementar.
