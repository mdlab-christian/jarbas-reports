⚠️ dangerouslySkipPermissions: true desabilita prompts de confirmação. Útil para automação — mas em MacBook pessoal, considere manter false se preferir mais controle.
7
Configurar MCPs
n8n MCP + 21st.dev Magic — mesmos da JANA

Os MCPs são configurados no ~/.claude.json. Edite o arquivo e adicione a seção mcpServers:

📄 ~/.claude.json — adicionar seção mcpServers
"mcpServers": {
  "n8n-mcp": {
    "type": "stdio",
    "command": "npx",
    "args": ["-y", "n8n-mcp"],
    "env": {
      "N8N_API_URL": "https://primary-production-e209.up.railway.app/api/v1",
      "N8N_API_KEY": "SEU_N8N_API_KEY_AQUI"
    }
  },
  "magic": {
    "type": "stdio",
    "command": "npx",
    "args": ["-y", "@21st-dev/magic@latest"],
    "env": {
      "API_KEY": "SEU_TWENTY_FIRST_API_KEY_AQUI"
    }
  }
}
ℹ️ Pegue as chaves certas no ~/jarbas/.env do Mac Mini (ou peça ao JARBAS). O N8N_API_KEY fica em N8N_API_KEY e o 21st.dev em TWENTY_FIRST_API_KEY.
Adicionar via CLI (alternativa mais simples)
# Adicionar MCP do n8n:
claude mcp add n8n-mcp \
  --command "npx" \
  --args "-y,n8n-mcp" \
  --env "N8N_API_URL=https://primary-production-e209.up.railway.app/api/v1" \
  --env "N8N_API_KEY=SEU_N8N_API_KEY"

# Adicionar MCP do 21st.dev:
claude mcp add magic \
  --command "npx" \
  --args "-y,@21st-dev/magic@latest" \
  --env "API_KEY=SEU_TWENTY_FIRST_API_KEY"
