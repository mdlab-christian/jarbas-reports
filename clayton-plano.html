    Criar usuário macOS clayton
    Com home em /Users/clayton, sem admin. Mesma abordagem do usuário olivia.
  • 📦
    Instalar OpenClaw no usuário clayton
    Copiar instalação ou instalar via npm com sudo -u clayton. Porta: 18795
  • ⚙️
    Configurar openclaw.json
    Baseado no config da OlivIA. Telegram bot token (aguardando), porta, workspace, model.
  • 🔑
    Configurar variáveis de ambiente
    Criar /Users/clayton/clayton/.env com ANTHROPIC_API_KEY, SUPABASE_KEY, chaves migradas
  • 🚀
    Criar serviço LaunchDaemon
    Autostart do gateway na porta 18795. Mesma abordagem do olivia-gateway.
