📋 Quando usar
Qual o critério para dar ciência com renúncia?▼
Quando a intimação é de decisão/despacho que NÃO exige manifestação ativa do escritório no prazo. Exemplos: despacho de mero expediente, decisão de gratuidade já concedida, intimação de distribuição de cumprimento que a outra parte deve cumprir. NÃO usar quando ainda há prazo para recurso ou manifestação que o escritório vai aproveitar.
Decisão desfavorável — devo dar ciência com renúncia?▼
SOMENTE se o escritório definitivamente não vai recorrer (decisão estratégica já tomada com o cliente). Se houver qualquer possibilidade de recurso, NÃO dar ciência com renúncia — aguardar decisão do cliente.
⚡ Passos para o JARBAS executar
1 Verificar se flag_ciencia_renuncia = true na análise IA
2 Confirmar que o escritório representa o polo correto (verificar advogado_id)
3 Entrar no EPROC via OAB correspondente → localizar processo → verificar intimação aberta → protocolar ciência com renúncia expressa ao prazo
4 Registrar número de protocolo → atualizar campo ciencia_renuncia_numero_protocolo

✅Múltiplas Atividades Sugeridas✨ Novo

⏱Tempo em Curso✨ Novo

📅Timeline de Fases✨ Novo

Visualização das fases do processo com duração de cada etapa. Calculada das movimentações. Fase atual destacada.
