🎭 Quem é o Clayton

O Agente de Controle Processual

Clayton é o nome do agente especializado em gestão processual do MP Advocacia dentro do MdFlow. Antes chamado de "Controller", foi renomeado para ter identidade própria dentro do ecossistema de agentes.

  • Gerencia todas as fases do processo judicial
  • Do cadastro à execução e pagamento
  • Substitui completamente o fluxo manual no Advbox
  • Integrado com JARBAS para automações via Telegram
  • Conectado ao EPROC (TJRS, TJSC, TJSP)

Herança do Advbox

Clayton absorveu e expandiu tudo o que o escritório fazia no Advbox — o software jurídico anterior. Com a API do Advbox, a transição é segura e rastreável.

  • Processos: número CNJ, fase, tipo, pasta, cliente
  • Movimentações: EPROC + manuais sincronizadas
  • Publicações: intimações capturadas automaticamente
  • Financeiro: honorários, custas, alvarás, acordos
  • Documentos: via export LGPD + organização no MdFlow

🚀 Plano de Migração Advbox → Clayton

Fase 0 — Acesso à API

Verificar plano atual ou solicitar API key ao suporte Advbox. Pedir export completo via LGPD (documentos).

Fase 1 — Importação base

GET /settings → mapear IDs. GET /customers → importar clientes. GET /lawsuits paginado → todos os processos.

Fase 2 — Enriquecimento

Detalhes financeiros por processo, movimentações completas, histórico de tarefas, publicações. Roda de madrugada (~8-12h com rate limit seguro).

Fase 3 — Validação paralela

Por N dias, comparação diária: intimações Advbox vs EPROC do JARBAS. Delta zero por 7 dias = seguro para cancelar.

Fase 4 — Descomissionamento

Cancelar plano Advbox. Clayton assume 100% do controle processual. OlivIA protocola petições automaticamente.
