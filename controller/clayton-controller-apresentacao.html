Triagem de Intimações

Análise e classificação automática

  • Identificar tipo de intimação por palavras-chave
  • Classificar por urgência e prazo
  • Detectar duplicatas e fechar automaticamente
  • Filtrar por OAB e tribunal no Advbox
  • Verificar se já existe tarefa em andamento
  • Distinguir intimações "de andamento" das que precisam de ação
  • Pauta de julgamento → sempre fechar, não precisa ação
  • Inclusão em pauta → fechar sem análise
📋

Regras de Decisão por Evento

O que fazer para cada tipo de decisão judicial

  • Sentença improcedente → criar tarefa IMPROCEDENTE 1º GRAU
  • Sentença procedente → criar tarefa JULGADO PROCEDENTE para chefes
  • Audiência instrução designada → DESISTÊNCIA (Yuri)
  • Perícia designada → DESISTÊNCIA (RS)
  • AJG + liminar deferida → registrar no histórico
  • Decisão 2º grau → criar tarefa DECISÃO 2º GRAU
  • Embargos declaração → criar tarefa MANIFESTAÇÃO
  • Indeferimento AJG → criar tarefa AGRAVO DE INSTRUMENTO
  • Liminar concedida (SISBAJUD) → EXPEDIÇÃO ALVARÁ (urgente)
  • Homologação de transação → Ciência + fechar
  • Cancelamento distribuição com custas → APELAÇÃO
  • Cancelamento distribuição sem custas → Ciência + arquivar
  • Extinção sem custas → Ciência + arquivar
  • Extinção com custas → APELAÇÃO
  • Provas/ato ordinatório → Ciência com renúncia ao prazo
  • Apelação provida → aguardar trânsito para cumprimento

Protocolo de Petições

EPROC TJRS — processo completo

  • Acessar processo no EPROC TJRS
  • Clicar em "Movimentar/Peticionar"
  • Selecionar motivo de prioridade (alvará, liminar, etc.)
  • Selecionar evento "PETIÇÃO"
  • Anexar PDF da petição via drag-and-drop ou file picker
  • Selecionar tipo de documento (PETIÇÃO, PED.EXP.ALV, etc.)
  • Confirmar seleção de documentos
  • Clicar em "Peticionar" e aguardar confirmação
  • Verificar evento gerado na lista de movimentações
  • Detectar e tratar erros de nomenclatura de arquivo
  • Petição fonte: Google Docs → exportar como PDF

TJRS — Sequência de Operação

1
Login EPROC TJRS

Fazer logout da OAB anterior, login com OAB correta (RS116571 ou RS131163)

2
Filtrar no Advbox

Intimações Judiciais → Que eu recebi → Pendentes → Grupo de ação = OAB

3
Para cada intimação

Copiar nº processo → abrir no EPROC → ler último evento → classificar

4
Executar ação correta

Ciência + renúncia OU criar tarefa com responsável + prazo

5
Fechar no Advbox

Marcar como Resolvido/Fechado. Fechar duplicatas sem análise.

6
Verificação final

Confirmar fila zerada. Anunciar conclusão e partir para próxima OAB.

🏛️ PJE — Justiça Federal SP
Primeiro treinamento no sistema federal — diferenças críticas
🔐
Principal barreira: Login com Certificado A3 + 2FA

O PJE exige token físico USB (certificado A3), PIN de 6 dígitos e código Google Authenticator. O app "PJE Office Pro" precisa estar ativo. Clayton conhece o fluxo completo mas a automação do login precisa de solução especial (integração com o token).

✅ Automatizável no PJE:

  • Copiar nº processo do Advbox para PJE
  • Navegar Menu → Processo → Pesquisar
  • Colar nº no campo de busca
  • Clicar no processo para abrir
  • Ler decisão no painel lateral
  • Criar tarefa "DEFERIDA LIMINAR" no Advbox
  • Fechar intimação no Advbox

❌ Requer solução especial:

  • Login com token USB físico A3
  • PIN de 6 dígitos do certificado
  • Código 2FA do Google Authenticator
  • Ativação manual do PJE Office Pro
  • Interpretação jurídica da decisão
  • Ciência com renúncia no PJE (botão ainda não documentado)
