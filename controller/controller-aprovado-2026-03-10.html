O que é o sistema Controller
🎛️

Central de Intimações

Página principal do MdFlow para gestão de todas as intimações jurídicas recebidas do EPROC (TJRS, TJSC, TJSP). Cada intimação passa por análise IA automática antes de chegar ao advogado.

🤖

JARBAS como Motor

O Mac Mini M4 (JARBAS v2.6.0) executa navegação Playwright no EPROC, captura intimações, aciona análise IA via Claude, e mantém o banco Supabase atualizado em tempo real.

⚡

Realtime End-to-End

Do momento em que o EPROC publica uma intimação até ela aparecer na tela do advogado com resumo IA: menos de 60 segundos. Tudo via Supabase Realtime + WebSockets.

📊

Dashboard Operacional

KPIs em tempo real: total não lidas, prazos vencendo hoje, backlog por advogado, taxa IA. Filtros compostos, paginação server-side de 50 itens, agrupamento por processo.

Funcionalidades completas

Tab Intimações — V3

Lista paginada (50/página server-side), agrupamento por processo, ordenação urgente→recente. Filtros: status, IA, prazo, responsável, empresa, flags.

Drawer lateral rico

5 tabs: Resumo IA, Processo, Histórico de movimentações, Anexos, Chat OlivIA. Redimensionável 38%–82% da tela. Shortcut "A" abre atividade direto.

KPIs em tempo real

Não lidas, prazos vencendo hoje, backlog IA, processadas. Banner vermelho animado quando há prazos para hoje com filtro automático.

Bulk Actions

Seleção múltipla com checkbox, barra flutuante: processar, ignorar, criar atividades em lote, transferir. Atualização via RPC batch.

Command Palette (Cmd+K)

Busca instantânea por CNJ ou cliente. Keyboard navigation j/k. Atalhos: ? mostra shortcuts, Escape fecha drawer.

Tab Atividades

Gestão de tarefas criadas a partir de intimações. Integrada com SheetAtividade para criação rápida com prazo e responsável.

Tab Documentos

Documentos pendentes de cliente. Solicitar via botão no drawer gera registro automático com notificação.

Tab Equipe (gestor)

Visão por advogado: backlog individual, produtividade, atividades abertas. Visível apenas para admin/gestor.

Tab Dashboard

Gráficos de distribuição, tendências, taxa IA por período. Integrado com dados reais do banco.

Tab Arquivados

Intimações processadas e ignoradas. Filtros e busca. Permite reativar se necessário.

Tab Monitoramento

Status JARBAS em tempo real (heartbeat), processos monitorados, skills ativas dinâmicas via metadata, agentes IA, webhooks n8n.

Tab Playbooks (admin)

Configuração de fluxos automatizados. Acesso restrito ao perfil admin.

Realtime em 3 canais

intimacoes (INSERT+UPDATE), jarbas_heartbeat (UPDATE), processos_monitoramento (ALL). Invalidação de cache automática no React Query.

Exportação

Exportar intimações filtradas via dialog com opções de formato. Respeita filtros ativos e total de resultados.

Error Boundaries

Cada tab isolada com ErrorBoundary. Se uma crashar, as outras continuam. TabErrorFallback com mensagem clara.

Chat OlivIA (graceful)

Tab no drawer para consulta IA sobre a intimação. Se webhook n8n não configurado, exibe mensagem clara de orientação em vez de erro.

Ciência com Renúncia (em breve)

Botão visível quando IA detecta flag_ciencia_renuncia. Desabilitado com tooltip "Em implementação" até skill EPROC estar pronta.

Automações JARBAS conectadas

skill_captura_intimacoes.mjs

Navega EPROC, identifica intimações por keywords, extrai texto de documentos, grava com signature_hash para dedup. Status inicial: nova + ia_status: pendente.

skill_monitorar.mjs + skill_monitorar_cron.mjs

Monitora 5.826 processos ativos via EPROC. Detecta novas movimentações, atualiza processos_monitoramento, aciona análise IA quando relevante.

skill_verificar_prazos_hoje.mjs ← NOVA

Criada hoje. Cron automático às 7h30 e 12h BRT. Busca intimações com data_prazo=hoje e notifica Telegram com lista formatada por processo e ação IA.

heartbeat.mjs

Envia sinal ao Supabase a cada 60s via RPC jarbas_heartbeat_upsert. Inclui: processos monitorados, movimentações do dia, erros, uptime, lista de skills ativas.

task-consumer.mjs

Consome fila jarbas_tarefas do Supabase. "Forçar Verificação" no frontend enfileira tarefa forcar_monitoramento que o consumer executa em segundos.

WH-CTRL-01 (n8n)

Webhook de ingestão ativo. Recebe payload de intimação, processa via Claude (análise jurídica completa), atualiza ia_status=concluido no Supabase.
