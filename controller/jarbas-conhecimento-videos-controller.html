Sistemas e Acesso — Mapeamento Completo
Como funciona cada sistema judicial
  • EPROC TJRS/TJSC: Login por OAB (usuário+senha). Fazer logout de uma OAB para logar em outra.
  • e-SAJ TJSP: Interface diferente, processos relacionados para apelação, lentidão frequente
  • PJE TRF3: Certificado A3 (token USB) + PJE Office Pro ativo + PIN 6 dígitos + 2FA Google Authenticator
  • EPROC às vezes dá "bug" no login — tentar novamente
  • TJSC bug de login: geralmente não aceita senha na primeira tentativa — normal, tentar de novo
  • Advbox: filtrar por Grupo de ação (OAB) + Intimações Judiciais + Pendentes
  • Intimações com tarefa existente → fechar sem analisar (regra de eficiência)

Protocolo de Petições EPROC
Skill nova — peticionamento automatizado
  • Petição gerada no Google Docs → Arquivo → Baixar → PDF
  • EPROC → processo aberto → "Movimentar/Peticionar"
  • Selecionar motivo de prioridade (alvará, liminar, urgente ou vazio)
  • Evento: sempre "PETIÇÃO" (autocomplete com "pet")
  • Anexar PDF: drag-and-drop ou "Escolher arquivo" → aguardar sinal verde "+"
  • Tipo de documento: PETIÇÃO (padrão) ou específico (PED.EXP.ALV, etc.)
  • "Confirmar seleção de documentos" → verificar tabela
  • Nomenclatura com caracteres especiais pode causar erro — usar nomes simples
  • "Peticionar" → verificar evento gerado na lista de movimentações
