🔧 Pendências e Bugs

  • 🔴
    Contrato ausente no Drawer — Restrição Principal P1 Bug

    Campo contrato não aparece na seção Restrição Principal. Afeta Aguardando Distribuição e Agendados.

  • 🔴
    Upload de documentos falhando P1 Bug

    Nenhum tipo de arquivo sobe. Erros: "Erro a enviar o contrato", "Erro a registrar comprovante".

  • 🔴
    Comprovante de Restrição travado em "gerando" P1 Bug

    Botão bloqueado indefinidamente. Cenário N8N existe — verificar integração. Deve usar orgao_restritivo correto.

  • 🟡
    Remover Comarca/Vara do modal Efetuar Distribuição P2 Melhoria

    Redundante: já existem Cidade, Vara e Foro Regional. Abre vazio.

  • 🟡
    Advogado: exibir só sigla + estado P2 Melhoria

    Drawer mostra nome completo. Deve ser apenas OAB/SP, OAB/RJ etc.

  • 🟡
    Drawer Aguardando Preparação: unificar componente P2 Melhoria

    Usar mesmo drawer. Botão "Enviar para Distribuição" (componente existente).

  • 🟡
    Dashboard: implementar novo gráfico P2 Feature

    Linhas com curva real, filtros OAB/Usuário, períodos, eixo X seg–dom, filtrar por data_distribuicao.

  • 🔵
    Investigar 2.780 em Aguardando Preparação P3 Investigação

    Empresa ré faltante em todos? Causa raiz após correções P1/P2.

  • 🔵
    Validar ligação Distribuição → Processos P3 Validação

    Após distribuido = true: some da UI? Aparece em Processos com todos os campos?
