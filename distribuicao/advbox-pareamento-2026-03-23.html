🔍 Diagnóstico do Problema

2 bloqueios identificados

❌ Bloqueio 1 — API Advbox retornando HTTP 403

A chave de API ADVBOX_API_TOKEN configurada no ~/jarbas/.env está retornando HTTP 403 Forbidden ao tentar chamar https://app.advbox.com.br/api/v1/lawsuits. Isso significa que mesmo que a automação esteja configurada, ela falharia silenciosamente.

⚠️ Bloqueio 2 — Sem automação de cadastro pós-distribuição

O workflow N8N-DIST-MANUAL (webhook de distribuição) não possui nenhum nó que chame a API do Advbox. Ele apenas chama a RPC rpc_efetuar_distribuicao_manual no Supabase e retorna o resultado. Não há step de cadastro no Advbox. Da mesma forma, as funções do banco (fn_efetuar_distribuicao, fn_trg_jarbas_distribuicao_lifecycle) não têm nenhuma chamada à API do Advbox.
