O que é o Claude Code?
Entenda antes de instalar

Claude Code é uma ferramenta de linha de comando criada pela Anthropic que coloca o Claude — uma das IAs mais avançadas do mundo — diretamente dentro do seu projeto. Diferente do ChatGPT ou do Claude.ai (site), aqui a IA tem acesso real aos seus arquivos e pode ler, escrever, criar e modificar código autonomamente.

✅ O que ele FAZ
  • Lê todos os arquivos do projeto
  • Escreve e edita código
  • Roda comandos no terminal
  • Corrige bugs automaticamente
  • Explica código existente
  • Cria projetos do zero
⚠️ O que ele NÃO é
  • Não é o Claude.ai (site)
  • Não é o ChatGPT
  • Não tem interface gráfica
  • Não é gratuito (cobra por uso)
  • Não funciona sem internet
💡 Como funciona o pagamento

Você paga por uso à Anthropic — normalmente alguns centavos de dólar por conversa. Para uso normal (desenvolvimento, perguntas, código), um saldo de $5–10 dura bastante. Não tem mensalidade obrigatória.

1
Instalar Node.js
Pré-requisito obrigatório — o Claude Code precisa do Node.js para funcionar
ℹ️ O que é Node.js?

Node.js é um ambiente que permite rodar JavaScript fora do navegador. O Claude Code é feito em JavaScript, então ele precisa do Node.js instalado. Pense como o "motor" que faz a ferramenta funcionar.

  • Acesse o site oficial do Node.js

    Abra o navegador e vá para: nodejs.org

  • Baixe a versão LTS

    Clique no botão verde escrito "LTS" (Long Term Support). Esta é a versão estável. Vai baixar um arquivo .msi.

  • Execute o instalador

    Dê duplo clique no arquivo baixado. Clique Next → Next → Next → Install. Não precisa mudar nada, as opções padrão funcionam.

  • Verifique a instalação

    Abra o Prompt de Comando (pressione Win + R, digite cmd, Enter) e rode:

Prompt de Comando (cmd)
node --version
# Deve aparecer algo como: v20.11.0

npm --version
# Deve aparecer algo como: 10.2.4
✅ Deu certo se...

Aparecerem números de versão (ex: v20.11.0). Se aparecer "não é reconhecido como comando interno", reinicie o computador e tente de novo.

2
Instalar o Claude Code
Um comando e está feito

Com o Node.js instalado, o Claude Code é instalado com um único comando. Abra o Prompt de Comando como Administrador:

⚠️ Importante: rodar como Administrador

Pressione Win, busque por cmd, clique com o botão direito em "Prompt de Comando" e escolha "Executar como administrador". Isso é necessário para instalar globalmente.

Prompt de Comando (Administrador)
npm install -g @anthropic-ai/claude-code

Aguarde o download completar (pode demorar 1–2 minutos dependendo da sua internet). Quando terminar, verifique:

Verificar instalação
claude --version
# Deve aparecer: @anthropic-ai/claude-code@x.x.x
💡 Se aparecer erro de política de execução

No Windows, às vezes o PowerShell bloqueia scripts. Se acontecer, rode este comando como Administrador no PowerShell (não no CMD):

Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

Confirme com S e tente instalar novamente.

3
Criar conta e pegar a API Key
Você precisa de uma chave para o Claude Code se conectar à IA
  • Acesse a plataforma Anthropic

    Vá para: console.anthropic.com no navegador.

  • Crie uma conta gratuita

    Clique em "Sign Up" e cadastre com seu e-mail. Confirme o e-mail quando chegarem a mensagem de verificação.

  • Adicione um método de pagamento

    Vá em Settings → Billing. Adicione um cartão de crédito. Você pode colocar um crédito mínimo (ex: $5). O uso é debitado conforme você usa — não tem cobrança automática mensal.

  • Crie a API Key

    Vá em API Keys → Create Key. Dê um nome (ex: "meu-pc") e clique em criar.

  • ⚠️ COPIE A CHAVE AGORA

    A chave aparece uma única vez. Começa com sk-ant-api03-.... Copie e guarde em lugar seguro (Bloco de Notas, etc.). Se fechar sem copiar, precisará criar outra.

⚠️ Nunca compartilhe sua API Key

Sua API Key é como uma senha bancária — quem tiver ela pode gastar seu crédito. Não coloque em repositórios públicos (GitHub), não mande no chat, não compartilhe com ninguém.

4
Configurar a API Key
Duas formas de fazer — escolha a que preferir

Você precisa informar a API Key para o Claude Code. Existem dois jeitos:

🏆 Método A — Variável de Ambiente (recomendado)

Configura uma vez e nunca mais precisa fazer de novo. A chave fica salva no sistema.

  • Abra as Variáveis de Ambiente do Windows

    Pressione Win + R, digite sysdm.cpl e Enter. Clique na aba "Avançado" → "Variáveis de Ambiente".

  • Crie uma nova variável do usuário

    Na seção de cima ("Variáveis de usuário"), clique em "Novo...".

  • Preencha os campos

    Nome: ANTHROPIC_API_KEY
    Valor: sk-ant-api03-SUA_CHAVE_AQUI

  • Clique OK e reinicie o terminal

    Feche o Prompt de Comando e abra de novo para a variável ser carregada.

🔵 Método B — Passar direto no terminal

Mais simples, mas precisa fazer toda vez que abrir um terminal novo.

PowerShell — passar a chave temporariamente
# No PowerShell:
$env:ANTHROPIC_API_KEY = "sk-ant-api03-SUA_CHAVE_AQUI"

# No CMD clássico:
set ANTHROPIC_API_KEY=sk-ant-api03-SUA_CHAVE_AQUI

Verificar se funcionou: Após configurar, abra um terminal novo e rode claude. Se aparecer o menu de boas-vindas, a chave está funcionando. 🎉

5
Primeiro uso — abrindo um projeto
Como navegar até seu projeto e começar a usar

O Claude Code funciona dentro de uma pasta. Ele lê todos os arquivos da pasta atual e entende o contexto do projeto. O fluxo é sempre:

Fluxo básico de uso
# 1. Navegue até a pasta do seu projeto
cd C:\Users\SeuNome\Desktop\meu-projeto

# 2. Inicie o Claude Code
claude

# 3. Pronto! Você estará no modo interativo
# Basta digitar o que quiser fazer
ℹ️ Como navegar entre pastas no Windows
  • cd Desktop — entra na pasta Desktop
  • cd .. — volta uma pasta
  • cd C:\caminho\completo — vai direto para um caminho
  • dir — lista o conteúdo da pasta atual
  • Dica rápida: No Windows Explorer, clique na barra de endereço, apague o caminho, digite cmd e pressione Enter — abre o terminal já na pasta certa!
💡 Alternativa: use o VS Code

Se você usa o Visual Studio Code, abra seu projeto nele, vá em Terminal → Novo Terminal. Você já estará na pasta certa automaticamente. Então é só rodar claude.

6
Comandos essenciais
Tudo que você vai usar no dia a dia
Comando O que faz
claude Inicia o modo interativo (principal — use sempre este)
claude "mensagem" Executa uma tarefa rápida sem modo interativo e sai
claude --print "msg" Executa, imprime o resultado e sai (útil em scripts)
claude --model opus Usa o modelo mais poderoso (mais caro, para tarefas complexas)
claude --model sonnet Modelo balanceado (padrão — bom para tudo)
claude --model haiku Modelo mais rápido e barato (para tarefas simples)
claude --version Mostra a versão instalada
claude --help Mostra todos os parâmetros disponíveis

8
Dicas avançadas
Para tirar o máximo da ferramenta
💡 Dica 1 — Seja específico nas instruções

Quanto mais detalhe você der, melhor o resultado. Em vez de "corrija o bug", diga "corrija o bug na função calcularTotal() no arquivo carrinho.js — ela está somando o frete duas vezes quando o pedido tem desconto".

💡 Dica 2 — Use o arquivo CLAUDE.md para contexto permanente

Crie um arquivo chamado CLAUDE.md na raiz do projeto. Tudo que você escrever nele, o Claude Code sempre vai ler antes de responder. É ótimo para explicar: como o projeto funciona, padrões de código que você usa, regras que ele deve sempre seguir.

Exemplo de CLAUDE.md
# Contexto do Projeto

## O que é
Sistema de e-commerce em React + Node.js.

## Regras de código
- Usar TypeScript com tipagem estrita
- Funções assíncronas sempre com async/await (nunca .then())
- Commits em português
- Nunca deletar arquivos sem confirmar comigo primeiro

## Arquivos importantes
- src/api/routes.ts — todas as rotas da API
- src/utils/helpers.ts — funções utilitárias
- .env.example — variáveis de ambiente necessárias
💡 Dica 3 — Peça para revisar antes de executar

Para mudanças importantes, adicione "mostre o que você vai fazer antes de executar" no final do pedido. Assim você pode revisar e aprovar antes de qualquer modificação.

💡 Dica 4 — Use /clear quando mudar de assunto

O Claude Code guarda o histórico da conversa para entender contexto. Quando terminar uma tarefa e for para outra completamente diferente, use /clear para limpar o contexto. Isso evita confusão e reduz custos.

💡 Dica 5 — Modo --print para automação

Você pode usar o Claude Code em scripts automatizados com o flag --print. Por exemplo, para gerar documentação automaticamente em cada commit.

✅ Economize dinheiro: escolha o modelo certo
  • Haiku (--model haiku) — perguntas simples, explicações rápidas
  • Sonnet (padrão) — desenvolvimento do dia a dia, bugs, código novo
  • Opus (--model opus) — arquitetura complexa, decisões difíceis, refatoração grande
9
Solução de problemas
Os erros mais comuns e como resolver
⚠️ Erro: "claude não é reconhecido como comando"

Causa: O npm não está no PATH do Windows, ou a instalação não foi como administrador.

Solução 1: Feche e reabra o terminal.
Solução 2: Reinstale como administrador.
Solução 3: Adicione manualmente ao PATH: C:\Users\SeuNome\AppData\Roaming\npm

⚠️ Erro: "Invalid API Key" ou "Authentication Error"

Causa: A API Key está errada ou não foi configurada.

Solução: Verifique se a variável ANTHROPIC_API_KEY está correta. No terminal, rode:
echo %ANTHROPIC_API_KEY% (CMD) ou echo $env:ANTHROPIC_API_KEY (PowerShell).
Deve aparecer sua chave. Se estiver vazia, refaça a configuração da seção 5.

⚠️ Erro: "Insufficient credits"

Causa: Seu saldo na Anthropic acabou.

Solução: Acesse console.anthropic.com → Billing → adicione mais crédito.

⚠️ Erro de política de execução no PowerShell

Causa: Windows bloqueando scripts por segurança.

Solução: Abra o PowerShell como administrador e rode:
Set-ExecutionPolicy RemoteSigned -Scope LocalMachine
Confirme com S.

⚠️ Claude modificou arquivo errado / fez algo indesejado

Prevenção: Use Git no projeto. Antes de tarefas grandes, faça git commit para salvar o estado atual. Se algo der errado, é só fazer git checkout . para voltar ao estado anterior.

Recomendação: Sempre use o Claude Code em projetos versionados com Git.
