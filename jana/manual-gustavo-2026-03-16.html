🔄 Comando migra — Handoff entre Agentes

Este é o poder da JARBAS Network. Você pode passar o contexto de uma conversa de um agente para o outro sem perder nada.

📋 Como usar
# JARBAS trava no meio de uma lapidação Jana, migra Jarbas # → JANA recebe o contexto e continua de onde parou
# JANA está ocupada, quer passar pro JARBAS Jarbas, migra Jana # → JARBAS pega o contexto e continua

O handoff inclui: projeto ativo, tarefas pendentes, arquivos abertos, decisões tomadas, links dos HTMLs gerados.

🧑‍💻 Pair Programming — Jana + Jarbas

Você pode usar os dois agentes juntos para trabalhar mais rápido. Um gera, o outro revisa e complementa.

Exemplos de uso em conjunto
# JARBAS gera o HTML, JANA dá opinião Jarbas, gera o relatório de /processos. Jana, depois complementa.
# JANA faz backend, JARBAS faz frontend Jana, implementa a query Supabase. Jarbas, atualiza o componente React.
# Code review cruzado Jana, revisa o que o Jarbas acabou de fazer.
