Funcionalidades Desbloqueadas
🔍
Busca Full-Text
Procurar qualquer termo dentro de todas as notas do vault. Retorna snippets com contexto e número de linha. Essencial para recuperar memória de sessões anteriores.
search-content
✍️
Criar Notas Estruturadas
Criar nota com path, conteúdo e template em um comando. Perfeito para salvar decisões, sessões, specs e checkpoints direto do Telegram.
create
🔄
Refactor Seguro de Links
Mover ou renomear uma nota atualiza todos os [[wikilinks]] do vault automaticamente. Sem links quebrados, sem manutenção manual.
move
🧠
Vault como Memória Viva
Salvar sessões, decisões e contexto da OlivIA diretamente no vault via comando. O Obsidian vira o cérebro persistente da JANA, além do MEMORY.md.
cérebro
📋
Listagem Rápida
Listar todas as notas de uma pasta do vault para verificar o que existe antes de criar ou editar. Evita duplicatas e mantém estrutura limpa.
list
🤖
Automação por Agente
Cada agente (JANA, Olivia) tem seu vault isolado configurado como default. Operações isoladas, sem risco de contaminar contextos entre agentes.
isolamento
Comandos que vou usar no dia a dia
# Buscar algo no vault (full-text) notesmd-cli search-content "decisão arquitetural OlivIA" → projeto-olivia/estado-atual.md (linha 42): "decisão arquitetural: usar Supabase..." # Criar nota de sessão direto do Telegram notesmd-cli create "projeto-olivia/sessoes/2026-03-25" --content "..." # Mover nota (links se atualizam automaticamente) notesmd-cli move "rascunhos/spec-v1" "projeto-olivia/specs/spec-v1" # Buscar por nome de nota notesmd-cli search "estado-atual" → projeto-olivia/estado-atual.md # Vault default configurado notesmd-cli print-default → mdlab-vault → /Users/jarbas/mdlab-vault
