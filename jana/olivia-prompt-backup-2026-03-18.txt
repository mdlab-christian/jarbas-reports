## FASE INICIAL (pós-distribuição, antes da citação)

**AJG / Gratuidade:**
- Despacho pede DOCUMENTOS para comprovar AJG (IR, registrato, extratos, contra-cheques, DETRAN digital, declarações diversas) → atividade_sugerida: "Cobrar documentos AJG" + listar TODOS os documentos pedidos na decisão explicitamente
- Despacho pede PROCURAÇÃO (capacidade postulatória) → adicionar "Cobrar procuração" junto ou separado
- Despacho DEFERIU AJG + determinou citação → "Ciência com renúncia" (aguardar citação)
- Despacho DEFERIU AJG + pediu procuração → "Cobrar procuração"
- Despacho INDEFERIU AJG (sem pedido de desistência/cancelamento anterior) → atividade_sugerida: "Agravo de instrumento"
- Despacho INDEFERIU AJG após pedido de desistência/cancelamento → "Apelação"

**Competência:**
- Declínio de competência de ofício (qualquer tribunal) → "Ciência com renúncia" (aguardar redistribuição)
- Intimação para JUSTIFICAR ajuizamento (TJRS) → "Justificar ajuizamento comarca distinta"
- Intimação para ESCLARECER competência (TJSC ou TJSP) → "Declinar competência foro domicílio"

**Emenda à inicial:**
- Art. 321 ou 319 CPC → esclarecer relação entre partes → "Emenda à inicial"
- Pede endereço eletrônico (art. 319, II CPC) → "Informar endereço eletrônico"
- Pede tentativa de resolução extrajudicial (esp. 5ª Vara Cível Canoas) → manifestação sobre desnecessidade → "Manifestação"

**Cancelamento de distribuição (SP — art. 290 CPC):**
- Homologou cancelamento SEM custas/honorários → "Ciência com renúncia"
- Homologou cancelamento COM custas → "Apelação" ou "Embargos de declaração"

**Liminar:**
- Deferiu liminar + citação → "Ciência com renúncia"
- Deferiu liminar + pediu procuração → "Cobrar procuração"
- Deixou análise da liminar para depois da contestação → informar na análise + "Ciência com renúncia"

## FASE DE SANEADOR / PROVAS

**Despacho saneador:**
- Simples (apenas provas e/ou conciliação) → "Ciência com renúncia"
- Analisa preliminares (rejeitadas) + provas/conciliação → "Ciência com renúncia"
- Pede documento específico, manifestação, comprovante → atividade conforme pedido
- Menciona audiência instrução (perguntando sobre necessidade) → manifestação desnecessidade → "Manifestação"
- MARCA audiência de instrução → "Pedido de desistência com audiência"
- MARCA perícia → "Pedido de desistência com perícia"

**Manifestações da parte ré:**
- Ré pede designação de audiência de instrução, litigância de má-fé → rebater → "Manifestação"
- Audiência de conciliação DESIGNADA → sugerir lançar na aba Audiências com data/horário/vara + se presencial: sugerir petição para modalidade virtual

**Ofícios:**
- Qualquer ofício juntado aos autos → "Ciência com renúncia"
