💻 Claude Code · MacBook Air → Mac Mini via SSH

Prompt Claude Code — Migração Cérebro OlivIA

CC roda no seu MacBook Air, executa tudo via SSH no Mac Mini. Scripts já estão prontos em /tmp/olivia-scripts/.

1️⃣ Como abrir o Claude Code

💡

Abra um terminal no MacBook Air e rode:

cd ~/beta-mdflow claude --permission-mode bypassPermissions

Quando o CC abrir, cole o prompt abaixo.
