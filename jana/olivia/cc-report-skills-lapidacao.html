5. server.mjs (NOVO)

Servidor HTTP do motor OlivIA CRIADO

Descrição

Servidor HTTP criado em /Users/Shared/olivia-server/server.mjs que orquestra o pipeline completo de skills OlivIA.

Rotas

MétodoRotaDescrição
POST/pipelinePipeline completo: analisa-provas → busca-juris → smartblocks → gera-perguntas (modo completo) ou → gera-peticao → revisor (modo express)
POST/gerar-peticaoGerar petição pós-perguntas. Extrai outline/secoes_selecionadas do payload e passa para gera-peticao
POST/skill/{nome}Invocar skill individual (analisa-provas, busca-juris, smartblocks, gera-perguntas, gera-peticao, revisor, cria-doc, pesquisa-web)
GET/healthHealth check
GET/skillsListar skills disponíveis

// E passam para gera-peticao.executar():

  ...dadosDoPipeline,
