🐛 Bugs Críticos (bloqueiam entrega final)
BUG-01 — GAS recebe payload errado: secoes é obrigatório
Onde: skill_olivia_agente_v2.mjs → finalizarGoogleDoc()
Causa: A skill chama GAS_URL_NOVO (Google Apps Script real) com { titulo_documento, peticao_html, tags }. Mas o GAS v4 espera { secoes: [{titulo, conteudo}] }.
Solução: Não chamar GAS diretamente. Redirecionar para http://mac-mini:3012/olivia/criar-doc (JARBAS) — exatamente como a skill legada faz.
O endpoint JARBAS aceita peticao_html + titulo_documento + google_folder_id e chama o olivia-doc-creator.mjs internamente.

// CORRETO — usar JARBAS /olivia/criar-doc (mesma interface que skill legada)

BUG-02 — peticao_status não existe em olivia_historico
Onde: skill_olivia_agente_v2.mjs → salvarHistorico()
Causa: SPEC menciona peticao_status: 'concluida', mas o schema real usa status com valores PENDENTE | SUCESSO | ERRO.
Solução: Remover campo peticao_status. Usar status: 'SUCESSO' (já está na skill, mas junto com o campo errado).
P0 
// ERRADO — campo não existe
peticao_status: 'concluida',

// CORRETO — remover essa linha; status já está correto
status: 'SUCESSO',
BUG-03 — Frontend chama /olivia/gerar-modo-agente no JARBAS, mas testa o server.mjs local (porta 18788)
Onde: useOliviaAgente.ts → gerarPeticao()
Causa: O hook chama JARBAS_BASE_URL/olivia/gerar-modo-agente (Mac Mini, porta 3012) — NÃO o server.mjs local. Mas o servidor local (server.mjs na porta 18788) tem um endpoint diferente (/olivia/gerar-agente). São dois sistemas paralelos sem integração.
Impacto: O teste via browser (localhost:8080) usa o fluxo JARBAS corretamente. O server.mjs é independente e só funciona para testes diretos (curl). Não é bug do frontend, mas causa confusão.
Ação: Decidir se o server.mjs local é mantido como fallback de JANA ou descontinuado. Se manter, separar claramente os dois fluxos.

// FIX-01: finalizarGoogleDoc — usar JARBAS /olivia/criar-doc
async function finalizarGoogleDoc(peticao_html, contexto, titulo) {
  const JARBAS_URL = process.env.JARBAS_BASE_URL || 'http://mac-mini:3012';
  const numeroCnj = contexto.processo?.numero_cnj || 'sem-cnj';
  const tituloDoc = titulo || `Petição - ${numeroCnj}`;

  try {
    const res = await fetch(`${JARBAS_URL}/olivia/criar-doc`, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json', 'mdflow': 'MdL1501@' },
      body: JSON.stringify({
        titulo_documento: tituloDoc,
        google_folder_id: '1JTbKJqbmPzRTLlqRYZEZ5pNY-w8r1ALZ',
        peticao_html,
        tags: {},
      }),
      redirect: 'follow',
      signal: AbortSignal.timeout(60000),
    });
    const json = await res.json();
    const docUrl = json.google_doc_url || json.doc_url || null;
    if (docUrl) log(`Google Doc criado: ${docUrl}`);
    else log(`criar-doc retornou sem URL: ${JSON.stringify(json).slice(0, 200)}`);
    return docUrl;
  } catch (err) {
    log(`finalizarGoogleDoc erro: ${err.message}`);
    return null;
  }
}
