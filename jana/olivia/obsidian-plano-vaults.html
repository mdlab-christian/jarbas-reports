🔧 Ferramenta: obsidian-headless (CLI oficial)
✅
obsidian-headless — CLI oficial da Obsidian para sync sem interface

Lançado recentemente (open beta). Usa a mesma API do Obsidian Sync, E2E encryption, sem precisar abrir o app. Perfeito para Mac Mini rodando headless.

npm install -g obsidian-headless # requer Node.js 22+ ob login # login na conta Obsidian (interativo) ob sync-list-remote # lista vaults remotos disponíveis ob sync-setup --vault "mdlab-vault" --path ~/mdlab-vault # conectar vault local ao remoto ob sync --continuous # sync contínuo (watch mode)

Comandos principais

  • ob login — autenticar conta Obsidian
  • ob sync-list-remote — ver vaults na nuvem
  • ob sync-setup --vault NAME — conectar vault
  • ob sync — sync one-shot
  • ob sync --continuous — watch mode 24/7
  • ob sync-config --mode pull-only — só recebe
  • ob sync-status — ver estado do sync

Modos de sync disponíveis

  • bidirectional — padrão, ambos leem e escrevem
  • pull-only — só recebe mudanças, ignora locais
  • mirror-remote — só recebe, reverte mudanças locais

Por vault:

  • mdlab-vault → bidirectional
  • olivia-vault → pull-only (no Mac Mini)
  • cecilia-vault → bidirectional
