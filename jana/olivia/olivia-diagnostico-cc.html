🔄 Step 2: Dois Canais Realtime

⚠️ Dois ouvintes Realtime no mesmo histórico

useOliviaWizard.ts abre um canal Realtime em olivia-wizard-{id} mas só ouve SUCESSO/CONCLUIDO/ERRO/AGUARDA_REVISAO.

💡 Na prática está funcionando. Mas considerar adicionar CONCLUIDO como evento que o canal do useOliviaWizard também ouve no Step 2, para unificar o controle.

```js
function sanitizarNomeArquivo(nome) {
  if (!nome) return "documento";
  // Remove padrões como CONTR7, PROC2, RG4, UUID-...
  return nome
    .replace(/\b(CONTR|PROC|RG|EXT|FAT|CDC)\d+\b/gi, '')
    .replace(/[a-f0-9]{8}-[a-f0-9]{4}-[a-f0-9]{4}-[a-f0-9]{4}-[a-f0-9]{12}/gi, '')
    .replace(/\.\w+$/, '')
    .replace(/^\d+[-_]?/, '')
    .trim() || "documento";
}
```
