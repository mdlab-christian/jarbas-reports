### FIX-05 [P1] — Header # RÉPLICA duplicado
Arquivo: /Users/Shared/olivia-skills/olivia-formata-peticao.mjs

O LLM re-gera o título no início do output, então aparece 2x.
Adicionar dedup de título no início da formatação:
```js
function removerTituloDuplicado(html) {
  // Remove segunda ocorrência de H1 se o mesmo texto aparece 2x nos primeiros 500 chars
  const h1Match = html.match(/]*>([^<]+)<\/h1>/);
  if (h1Match) {
    const titulo = h1Match[1];
    const escapado = titulo.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
    const regex = new RegExp(`(]*>${escapado}<\/h1>)(?=.*\\1)`, 'gs');
    return html.replace(regex, '');
  }
  return html;
}
```
