Input: { modelo_grupo_id, assunto_ids[], uf_processo, limite: 10 }

Output: { juris: [{id, ementa, tribunal, data, texto_integral, embedding_similarity}] }

  3. Se < 3 resultados por UF → fallback: outras UFs
  4. Ordenar por embedding_similarity com o contexto do processo (fn_rag_search)
  5. Retornar top N mais relevantes
