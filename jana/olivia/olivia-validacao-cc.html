P0-01: n8n Express — 13 Code nodes usam fetch()
fetch() NAO existe no sandbox n8n 2.36. Todos os 13 Code nodes do workflow Express (MN5kuLDzuIrIpufm) vao falhar com ReferenceError: fetch is not defined. Sem esse fix, o Modo Express inteiro nao funciona.
Fix validado E2E (2026-03-19): 
// ANTES (quebrado):
const response = await fetch(url, options);

// DEPOIS (validado):
const https = require('https');
function httpRequest(options, body) {
  return new Promise((resolve, reject) => {
    const req = https.request(options, (res) => {
      let data = '';
      res.on('data', chunk => data += chunk);
      res.on('end', () => resolve({ statusCode: res.statusCode, body: data }));
    });
    req.on('error', reject);
    if (body) req.write(body);
    req.end();
  });
}

Esforco: ~2h (13 nodes, cada um precisa rewrite manual). Testado contra Supabase GET/PATCH, Anthropic API e GAS.
