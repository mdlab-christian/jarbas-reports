🎯 Visão Geral do Projeto
Reconstrução completa da página OlivIA — Gerador de Petições 2.0
🎯Objetivo

Reconstruir do zero a página OlivIA com 3 modos de geração, arquitetura resiliente, UX premium e 100% funcional. A página mais importante do MdFlow.

🛡️Princípio de Resiliência

JARBAS pode falhar (tecnologia nova). O escritório não pode parar. Cada modo é independente: Smart Blocks → N8N → JARBAS. Hierarquia de fallback garantida.

⚡Diferencial Técnico

Só trabalhamos com ações de massa do mesmo tipo de processo. Todo RAG, todo conhecimento é especializado. Isso eleva muito a qualidade vs concorrência.

💡 Regra de Ouro: A IA NUNCA inventa fundamentações. Usa APENAS conhecimento interno (RAG + banco). Segue sempre a linha do modelo de petição como esqueleto.

🏷️ Hierarquia Tags / Blocos de Texto
A espinha dorsal do sistema. Deve ser mantida em TODOS os modos.
Regra Mestre — Tag = Pergunta, Bloco = Resposta
Modelo de Petição — Esqueleto base. Sempre usado, mesmo no Agente.
↓ tem
Tags — Tópicos/Perguntas. "O que falar sobre restrição indevida?"
↓ respondidas por
Blocos de Texto — Conteúdo pré-escrito. "Texto da argumentação sobre restrição indevida."
↓ agrupados por
Versão (V1, V2...) — Por advogado. Cada versão = seleção diferente de blocos.
↓ complementados por
Jurisprudências — Estado preferencial. Fallback: mais recente do assunto.
Como a IA Usa Essa Hierarquia
  • 📥 Recebe: modelo com tags vinculadas
  • 🔍 Analisa: provas + dados do processo
  • 🧠 Decide: para cada tag, qual bloco usar (ou cria novo)
  • 🔄 Pode: reposicionar tags, criar tags novas, excluir irrelevantes
  • 🚫 NUNCA: inventar conteúdo sem base no RAG/banco
  • 📤 Responde: JSON com 1 item por tag → cola na petição
// Resposta da IA em JSON { "tags_resolvidas": [ { "tag_id": "uuid-tag-restricao", "tag_nome": "Restrição Indevida", "ordem": 1, "conteudo": "A restrição de crédito...", "bloco_base_id": "uuid-bloco", "modificacoes": "Ajustado para valor R$ 5.430,00" } ], "tags_criadas": [ { "nome": "Dano Moral Majorado", "conteudo": "...", "justificativa": "Identificado reincidência nas provas" } ] }
