Problema resolvido

O hook antigo (useExpressGeneration) lia state.processoId, state.modeloId etc. dentro de closures assincronos. Como updateState era chamado e imediatamente depois dispararAnalise() via setTimeout(100), o React ainda nao tinha propagado o state — a closure via os valores null anteriores.

Fix: iniciarAnalise(dados) recebe todos os dados como parametro. dispararGeracao le de dadosRef.current e historicoIdRef.current (refs, nao state).
