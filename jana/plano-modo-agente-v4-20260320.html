03 — Chat OlivIA via Edge Function (Supabase)

Por que Edge Function para o chat?

⚡
Latência mínima
Edge Functions rodam nos servidores Supabase, mesmo datacenter que o banco. Ler olivia_messages = menos de 5ms.
🔒
Auth nativo
Supabase JWT automático — a EF já sabe quem é o usuário logado. Zero trabalho de autenticação.
📡
SSE streaming nativo
Deno suporta streaming de resposta. Claude API → stream → SSE para o browser. Chat fluido.
🛡️
Independente do Mac Mini
Se JARBAS estiver offline, o chat ainda funciona. OlivIA responde via Claude API direta.

Arquitetura da Edge Function olivia-chat

// supabase/functions/olivia-chat/index.ts
import { serve } from 'https://deno.land/std/http/server.ts'
import { createClient } from '@supabase/supabase-js'
import Anthropic from 'npm:@anthropic-ai/sdk'

serve(async (req) => {
  // 1. Auth via JWT Supabase
  const supabase = createClient(...)
  const user = await getUser(req, supabase)
  
  // 2. Parse body
  const { processo_id, message, org_id } = await req.json()
  
  // 3. Upsert thread por processo
  const thread = await upsertThread(supabase, processo_id, org_id)
  
  // 4. Salvar mensagem do usuário
  await saveMessage(supabase, thread.id, 'user', message)
  
  // 5. Buscar histórico + contexto do processo
  const history = await getHistory(supabase, thread.id, 20)
  const processo = await getProcesso(supabase, processo_id)
  
  // 6. Chamar Claude com SSE streaming
  const stream = await claude.messages.stream({
    model: 'claude-sonnet-4-6',
    system: buildSystemPrompt(processo),
    messages: buildMessages(history, message),
    max_tokens: 2048,
  })
  
  // 7. Stream SSE de volta + salvar resposta
  return streamSSEResponse(stream, supabase, thread.id)
})

import { google } from 'googleapis';
import { JWT } from 'google-auth-library';

function getAuth() {
  const credentials = JSON.parse(
    process.env.GOOGLE_SERVICE_ACCOUNT_JSON
  );
  return new JWT({
    email: credentials.client_email,
    key: credentials.private_key,
    scopes: [
      'https://www.googleapis.com/auth/drive',
      'https://www.googleapis.com/auth/documents',
    ],
  });
}
