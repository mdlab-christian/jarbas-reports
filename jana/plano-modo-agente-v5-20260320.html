Problema 2 — DNS não configurado

Hostname jana.jarbas-mdlab.com precisa existir no Cloudflare Dashboard (Zero Trust → Tunnels → jana → Public Hostnames)

Problema 3 — Tunnel não é serviço

Tunnel não instalado como LaunchAgent → não reinicia com o sistema. Precisa de cloudflared service install

Sequência de fix (terminal MacBook Air)

# 1. Corrigir porta no config.yml
sed -i '' 's|http://localhost:18788|http://localhost:18789|g' ~/.cloudflared/config.yml
cat ~/.cloudflared/config.yml  # confirmar

# 2. Testar tunnel manualmente (verificar se o erro era porta ou DNS)
cloudflared tunnel run jana 2>&1 &
sleep 5 && curl -s http://localhost:18789 | head -20

# 3. Se DNS não propagou — adicionar no Cloudflare Dashboard:
# dash.cloudflare.com → Zero Trust → Networks → Tunnels → 4fdb4387 → Edit → Public Hostnames
# Subdomain: jana | Domain: jarbas-mdlab.com | Service: http://localhost:18789

# 4. Instalar como serviço (reinicia automático)
sudo cloudflared service install
sudo launchctl start com.cloudflare.cloudflared

# 5. Verificar
cloudflared tunnel info jana
curl -s https://jana.jarbas-mdlab.com  # deve responder
