O que é esta campanha

A Campanha de Acordos SP é uma iniciativa para contatar escritórios adversários nos processos da OAB SP535101 (Christian Paz, TJSP) e propor acordos extrajudiciais de forma ativa e organizada.

O sistema judicial paulista utiliza o e-SAJ — diferente do EPROC usado no RS/SC — o que exige scripts próprios para automação (a Cecília não serve para SP).

Por que fazemos isso

  • Gerar receita rápida via acordos antes de sentença
  • Critério do João: processos com contestação + réplica pendente ou aguardando sentença
  • Evitar processos onde já houve sentença ou desistência

Script Playwright v10 — Status

✅ Fase 1: textos dos 4 itens ❌ Fase 2: cores não aplicadas /tmp/sheets_v10.mjs

Problema identificado

O botão OK do color picker do Google Sheets usa display: none quando inativo, e não offsetParent: null. O script v10 usa a verificação errada.

Fix para v11

// v10 (bugado) — não detecta o botão OK corretamente
const btn = buttons.find(b => b.textContent.trim() === 'OK' && b.offsetParent !== null);

// v11 (correto) — usa getComputedStyle para detectar visibilidade
const btn = buttons.find(b =>
  b.textContent.trim() === 'OK' &&
  getComputedStyle(b).display !== 'none'
);
⚠️
Se preferir aplicar cores manualmente no Sheets: selecionar range de uma cor → Format → Conditional formatting → "Text is exactly" → inserir o valor → escolher a cor de fundo listada acima.

async function enrich(cnj) {
  console.log(`\n🔍 Enriquecendo: ${cnj}`);

  const agent = await client.beta.agents.create({
    name: 'campanha-enricher',
    model: 'claude-opus-4-6',
    system: SYSTEM,
    tools: [{ type: 'agent_toolset_20260401' }],
  });

  const environment = await client.beta.environments.create({
    name: 'campanha-env',
    config: { type: 'cloud', networking: { type: 'unrestricted' } },
  });

  const session = await client.beta.sessions.create({
    agent: agent.id,
    environment_id: environment.id,
    title: `Enrich ${cnj}`,
  });

  const stream = await client.beta.sessions.events.stream(session.id);

  await client.beta.sessions.events.send(session.id, {
    events: [{
      type: 'user.message',
      content: [{ type: 'text', text: `Busque os dados de contato do advogado réu do processo: ${cnj}\nRetorne apenas o JSON.` }],
    }],
  });

  let result = '';
  for await (const event of stream) {
    if (event.type === 'agent.message') {
      for (const block of event.content ?? []) {
        if (block.type === 'text') result += block.text;
      }
    }
    if (event.type === 'agent.tool_use') process.stdout.write(`  [${event.name}]\n`);
    if (event.type === 'session.status_idle') break;
  }

  // Extrair JSON do resultado
  const jsonMatch = result.match(/\{[\s\S]*\}/);
  return jsonMatch ? JSON.parse(jsonMatch[0]) : { cnj, status: 'erro', raw: result };
}

// Rodar
