Cecília faz o que humano não aguenta

Abrir 100 processos no EPROC, um por um, navegar até a contestação e copiar o texto levaria dias. Cecília faz isso em 3–5 horas no Mac Mini, enquanto você dorme.

OlivIA entende o que Cecília lê

Cecília retorna texto bruto da contestação. OlivIA lê isso, identifica quem é o advogado, qual o escritório, e extrai os dados de forma estruturada — como um advogado faria.

Claudio encontra o que não está no processo

Contestação nem sempre tem e-mail do escritório. Claudio navega no site da OAB, Google e LinkedIn para encontrar o contato — como o Gustavo faria, só que sem precisar do Gustavo.

💡
Por que não usar a Cecília para tudo?

Cecília é especializada em EPROC — ela não navega sites externos como OAB, Google ou LinkedIn. Claudio (Computer Use) foi feito para isso: navegar qualquer site da internet. São ferramentas diferentes para tarefas diferentes. A combinação das duas cobre todo o fluxo sem pontos cegos.

🏢 Por que agrupar por escritório?

Um mesmo escritório como Egger e Mesquita pode representar a empresa em 10, 20 ou 30 processos nossos. Em vez de mandar 30 mensagens individuais, a IA:

  • ✅ Agrupa todos os processos pelo mesmo escritório
  • ✅ Envia 1 mensagem mencionando todos os processos
  • ✅ Propõe negociação em bloco — mais eficiente
  • ✅ Aumenta chance de resposta (é mais profissional)

📁 Novo Script — campanha_acordos.mjs (Mac Mini)
// campanha_acordos.mjs — Roda no Mac Mini via SSH
// Reutiliza: skill_login_eproc, skillAcessarProcesso, skillLerDocumento
// Novo: lógica de busca de contestação + análise Claude

import { skillLoginEproc } from './skills/skill_login_eproc.mjs';
import { skillAcessarProcesso } from './skills/skill_acessar_processo.mjs';
import { skillLerDocumento } from './skills/skill_ler_documento.mjs';
import { supabase } from './lib/supabase.mjs';
import Anthropic from '@anthropic-ai/sdk';

const CONTESTACAO_PATTERNS = [
  'CONTESTAÇÃO', 'CONTESTACAO', 'JUNTADA DE CONTESTAÇÃO',
  'APRESENTOU CONTESTAÇÃO', 'CONTESTAÇÃO PROTOCOLADA',
  'RESPOSTA DO RÉU'
];

async function rodarCampanha(cnjs) {
  const { page, sessionHash } = await skillLoginEproc({ tribunal: 'TJSP' });

  for (const { id: processoId, cnj } of cnjs) {
    try {
      // 1. Acessar processo
      const { eventos } = await skillAcessarProcesso(page, {
        cnj, sessionHash, tribunal: 'TJSP', maxEventos: 50
      });

      // 2. Encontrar contestação
      const eventoContest = eventos.find(e =>
        CONTESTACAO_PATTERNS.some(p =>
          e.texto?.toUpperCase().includes(p))
      );

      if (!eventoContest?.docHref) {
        await upsert(processoId, { status: 'sem_contestacao' });
        continue;
      }

      // 3. Ler documento
      const { texto } = await skillLerDocumento(page, {
        docHref: eventoContest.docHref, cnj
      });

      // 4. Extrair dados com IA
      const dados = await extrairDadosAdvogado(texto);

      // 5. Salvar
      await upsert(processoId, {
        status: 'extraindo',
        contestacao_texto: texto,
        ...dados,
        contato_fonte: 'eproc_contestacao'
      });

    } catch(e) {
      console.error(`Erro ${cnj}: ${e.message}`);
    }
  }
}
