3. Skills Disponiveis (11 total)

OlivIA — 5 skills

/gerar-peticao
Wizard completo: CNJ → docs → analise → tags → geracao → vibelaw → relatorio. Fluxo guiado passo a passo.
/analisar-processo
Analisa todos os documentos de um processo, extrai fatos relevantes, teses aplicaveis e pontos de atencao.
/buscar-juris
Busca jurisprudencia por query livre. Retorna decisoes relevantes com tribunal, data e ementa resumida.
/lote-peticoes
Geracao em lote para campanhas (acordos Serasa, replicas padrao). Processa multiplos CNJs sequencialmente.
/treinar-olivia
Registra feedback sobre peticao gerada. Salva em vault/pendentes/ para revisao do Christian.

Cecilia — 5 skills

/status-cecilia
Saude do sistema: ultima captura, OABs ativas, pendencias, erros recentes e fila de processamento.
/prazos
Deadlines da semana com marcacao de urgencia. Destaque para D-0 (hoje) e D-1 (amanha). Filtravel por OAB.
/capturar [rs|sc|sp]
Forca captura EPROC para estado especifico. Respeita delays anti-bot (800-2500ms entre processos).
/relatorio
Relatorio diario consolidado: capturas, intimacoes novas, classificacoes, erros e metricas de cobertura.
/treinar-cecilia
Correcao de classificacao de intimacao. Reclassifica taxonomia A-G e salva para aprendizado do sistema.

Geral — 1 skill

/ajuda
Menu completo com todos os comandos disponiveis, descricoes e exemplos de uso para cada skill.
