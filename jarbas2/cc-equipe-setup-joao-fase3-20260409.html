Secao 1 — O que o Joao precisa fazer AGORA (Terminal)

▼
Como abrir o Terminal

Pressione Cmd + Espaco no teclado, digite Terminal e aperte Enter.

Vai abrir uma janela preta/branca com texto. E ali que voce vai colar os comandos abaixo.

1 Verificar o que ja esta instalado

Cole CADA comando abaixo no Terminal, um por vez, e mande o resultado pro Christian:

# 1. Versao do macOS sw_vers
# 2. Homebrew instalado? brew --version
# 3. Node.js instalado? node --version
# 4. npm instalado? npm --version
# 5. Claude Code instalado? claude --version
# 6. Seu IP na rede local ifconfig en0 | grep "inet " | awk '{print $2}'
# 7. SSH esta habilitado? sudo systemsetup -getremotelogin
# 8. Seu nome de usuario whoami
# 9. Seu hostname hostname
⚠️
Sobre o comando 7 (SSH)

Esse comando vai pedir a senha do Mac (a mesma que voce usa pra desbloquear).

Quando voce digitar a senha, nada vai aparecer na tela — isso e normal! Digita e aperta Enter.

2 Se Node.js NAO estiver instalado

Se o comando node --version deu erro "command not found", rode:

brew install node
⏳
Pode demorar 10-15 minutos

No macOS 12 Intel, o brew install pode levar um tempo. NAO feche o Terminal.

Voce vai ver varias linhas passando — e normal. Espere ate aparecer o simbolo $ de novo.

3 Se Claude Code NAO estiver instalado

Se o comando claude --version deu erro e o Node ja esta ok:

npm install -g @anthropic-ai/claude-code
4 Verificar que tudo esta ok

Rode esse comando e mande o resultado pro Christian:

node --version && npm --version && claude --version
Resultado esperado (exemplo)
v22.12.0 10.9.2 1.0.16

Os numeros podem ser diferentes — o importante e que os 3 aparecam sem erro.

▼

Secao 3 — Gerar chave SSH para o Mac Mini

▼
Por que isso e necessario?

A chave SSH permite que o seu Mac se conecte de forma segura ao Mac Mini do escritorio (onde ficam os dados e automacoes). E como uma "senha digital" unica do seu computador.

7 Gerar a chave SSH

Cole esse comando. Ele vai criar dois arquivos (chave publica + privada):

ssh-keygen -t ed25519 -f ~/.ssh/cc-equipe-ed25519 -N "" -C "joao@cc-equipe"
Resultado esperado

Voce vai ver algo como "Your identification has been saved" e um desenho feito de letras. Isso e normal.

8 Copiar a chave publica e mandar pro Christian

Esse comando vai mostrar a chave publica na tela. Copie TODO o texto e mande pro Christian:

cat ~/.ssh/cc-equipe-ed25519.pub
📋
IMPORTANTE: Copie TUDO

O resultado vai ser uma linha longa comecando com ssh-ed25519 AAAA...

Selecione TUDO (clique triplo ou Cmd+A no Terminal), copie (Cmd+C) e mande pro Christian via WhatsApp.

9 Configurar acesso ao Mac Mini

Cole esse bloco INTEIRO de uma vez no Terminal:

mkdir -p ~/.ssh && cat >> ~/.ssh/config << 'EOF' Host mac-mini HostName 100.78.173.81 User jarbas IdentityFile ~/.ssh/cc-equipe-ed25519 ConnectTimeout 10 ServerAliveInterval 30 EOF
O que isso faz?

Cria um "atalho" chamado mac-mini para que o Claude Code possa se conectar ao servidor automaticamente. Voce nao precisa decorar IPs.

Notas Importantes — Leia antes de comecar

▼
🚨
Se algum comando der "command not found"

brew nao encontrado → precisamos instalar Homebrew primeiro

node nao encontrado → precisa instalar via brew (Passo 2)

claude nao encontrado → precisa instalar via npm (Passo 3)

Mande um screenshot do erro pro Christian — ele vai te dizer o que fazer.

🔒
Se pedir senha de admin do Mac

E a senha que voce usa pra desbloquear o Mac.

Quando voce digitar, nada aparece na tela (nem bolinhas) — isso e normal e e por seguranca.

Digita a senha e aperta Enter.

🚫
NAO feche o Terminal

Enquanto um comando estiver rodando (especialmente brew install), NAO feche o Terminal.

Voce sabe que o comando terminou quando aparece uma nova linha com o simbolo $ ou % e o cursor piscando.

Se o Homebrew nao estiver instalado

Se brew --version der erro, rode esse comando para instalar:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Isso pode demorar 15-20 minutos. Vai pedir a senha do Mac. NAO feche o Terminal.
