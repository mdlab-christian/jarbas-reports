🔄 Workflow Definitivo (3 Ciclos/Dia)
Ciclos: 00:00 (meia-noite) · 12:00 (meio-dia) · 18:00 (18h) — janela de 6h entre cada ciclo
1

Login + Sessão MANTER

skill_login_eproc.mjs — SSO Keycloak + 2FA TOTP + seleção de perfil ADVOGADO. Roda para cada OAB do ciclo.

2

Captura Lista de Movimentações NOVA SKILL

Clica em "Últimas Movimentações" (link do painel com hash). Extrai todos os CNJs com movimentação na janela de tempo (últimas 6h). Pagina automaticamente. Retorna lista de CNJs.

  • Filtro por data: apenas movimentações desde o último ciclo
  • Coleta: CNJ, data/hora, descrição do último evento
  • Volume esperado: ~200-300 por ciclo
3

Processar Processo por Processo NOVA SKILL

Para cada CNJ da lista: clicar no número do processo (link com hash). Verificar se movimentação é para parte autora. Se sim: identificar evento + clicar no documento + extrair texto.

  • Filtro parte autora: descartar movimentações da parte contrária
  • Acesso ao documento: clique direto (não URL)
  • Fallback: Árvore do processo se documento não acessível diretamente
  • Fallback Vision: se Playwright falhar (5% dos casos)
  • Re-login automático se sessão expirar no meio
4

Análise IA MANTER

skill_analise_intimacoes_bulk + RAG com playbooks do João. Claude Sonnet 4.6 analisa o texto extraído, aplica playbook, classifica (A-G), define prazo e ação sugerida.

5

Salvar + Notificar MANTER

Gravar resultado no Supabase (tabela intimacoes: conteudo, ia_analise, ia_status, prazo, acao_sugerida). Notificar João via Telegram com resumo do ciclo.

6

Ciência Automática (opcional) A DEFINIR

Para casos onde João definiu ciência automática: dar ciência no EPROC. Sempre com confirmação para casos novos.

🛡️ Arquitetura de Fallback (à prova de erro)
Camada 1 — Playwright

95% dos casos. Rápido, barato, determinístico. Nunca usa URL direta — sempre clique no elemento.

Camada 2 — Playwright + retry

Falha no selector? Retry 3x com waitForLoadState. Sessão expirou? Re-login automático + continua do ponto que parou.

Camada 3 — Vision fallback

Selector ainda quebrado? Screenshot → Claude vision analisa a tela → extrai novo selector → tenta de novo. ~$0.05 por inspeção.

Regra inviolável: Se depois de todos os fallbacks ainda falhar → registrar no Supabase como status='falha_manual' + notificar João no Telegram. Nunca perder prazo silenciosamente.
