10-60s por processo

Read Docs
30-120s por proc
Pre-triagem regex: 60-70% e ruido obvio — pular leitura de doc. Para eventos relevantes: extrai link PDF, abre documento, extrai texto (timeout + retry). Fallback: Arvore se conteudo vazio.
S6
Analyze (IA)
10-17s por proc
RAG: top-5 playbooks similares (embeddings). Monta prompt: SOUL + playbooks + exemplos. Sonnet 4.6: tipo_acao + urgencia + prazo + flags. Schema validator. Retry 2x (5s, 15s).
S7
Persist
2-5s por proc
UPDATE intimacoes: todos os campos de uma vez (tipo_acao, analise_ia, modelo_ia, custo_ia). Registra custo em cecilia_custo_ia. Marca ciclo_item como processado.
S8
Act
Imediato para urgentes
urgente_financeiro/prazo_recursal → Telegram Joao imediato. ciencia_simples → fila /controller (1-click). peticao_necessaria → sugestao atividade. ruido → auto-finaliza silencioso.
S9
Report
~30s
Telegram para Joao: N processos, N urgentes, N erros, $X custo. Atualiza cecilia_ciclos: status, metricas. Registra em jarbas_heartbeat: sinal de vida do daemon.
