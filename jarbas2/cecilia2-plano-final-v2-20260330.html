🔍 Gaps Encontrados + Soluções

⚡
GAP 1 — TOTP condicional

Login às vezes pede TOTP (2FA), às vezes não. A skill_login atual falha quando não aparece.

✅ Solução: Detectar presença do campo TOTP após login. Se aparecer → preencher. Se não → ignorar. Timeout de 3s para esperar.

📄
GAP 2 — Documentos PDF não têm texto via innerText

PDF embarcado no browser viewer. innerText pode não funcionar. Confirmado ao vivo: PDF tem texto selecionável.

✅ Solução: Tentar via evaluate innerText do frame iframe. Fallback: baixar PDF com fetch() + credenciais, usar pdf-parse em Node.js. PDFs são texto, não scan — sem necessidade de OCR.

🔄
GAP 3 — Playwright via sudo no Mac Mini

SIGTRAP no Chrome headless quando executado via sudo interativo. Gatekeeper bloqueia.

✅ Solução: Skills rodam via LaunchAgent da cecilia (processo nativo, não sudo). Quando Jarbas dispara via SSH para executar skill → usar launchctl kickstart ou openclaw run skill como cecilia diretamente.

🔑
GAP 4 — Hash expira com sessão (~15min)

sessionHash obtido no login é inválido se sessão expirou. Erro: "hash inválido".

✅ Solução: cecilia_session_cache com TTL 14min (conservador). Antes de cada request, verificar se cache é válido. Se não → re-login automático. Não usar hash do último ciclo.

📊
GAP 5 — XLS não tem href de processo

A planilha XLS tem CNJ, evento, data, etc — mas NÃO tem o href com hash do processo. Precisa navegar pela lista HTML para pegar os hrefs.

✅ Solução: Fluxo correto é: (1) baixar XLS para filtrar quais processos analisar; (2) para cada CNJ selecionado, buscar via campo de pesquisa do EPROC → capturar href. OU: usar XLS só para data e evento, e para cada processo relevante fazer busca direta por CNJ no campo de pesquisa do EPROC.

🎯
GAP 6 — Último evento da lista vs. último evento do processo

A lista "Últimas Movimentações" ordena por data do último evento. O evento mostrado é o último. Mas o processo pode ter o documento em outro evento (o vinculado é ao evento específico).

✅ Solução: Para cada processo, acessar a página do processo completa e analisar apenas o(s) evento(s) novos (desde último ciclo). Guardar última data analisada em cecilia_ciclos para delta.

💰
GAP 7 — RS131163 (Heloísa) sem credenciais

Credenciais não estão em jarbas_accounts. OAB de Heloísa não vai funcionar.

✅ Solução: Inserir credenciais manualmente antes da Etapa 8. Jarbas pergunta ao Christian as credenciais antes de reativar.
