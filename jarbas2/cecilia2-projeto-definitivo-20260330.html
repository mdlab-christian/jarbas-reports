Penhora SISBAJUD, bloqueio conta, leilão, cumprimento de sentença com prazo. → Notificar imediatamente, ação em 24h.

CLASSE B — Decisão Judicial com Prazo

CLASSE C — Audiência/Perícia Agendada

Designação de audiência, perícia marcada. → Criar lembrete + verificar procuração.

CLASSE D — Decisão Informativa

Despacho de juntada, decisão interlocutória simples. → Registrar histórico MdFlow.

CLASSE E — Marco Processual

Sentença, acórdão, trânsito em julgado. → Registrar com destaque, aguardar instrução.

CLASSE F — Alvará/Levantamento

Alvará expedido, depósito judicial disponível. → Ação imediata de levantamento.

CLASSE G — Ruído (descartar)

DJE, DJM, remessa/recebimento autos, concluso, petição protocolada, intimação ré. → Zero custo IA.
