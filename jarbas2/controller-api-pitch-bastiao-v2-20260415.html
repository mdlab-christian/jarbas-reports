CONTROLLER API

A Infraestrutura
Que Alimenta Tudo.

Intimações, documentos e decisões judiciais capturados, lidos e classificados automaticamente. Delay de horas, não de dias. Análise de IA inclusa, sem custo adicional.

S1 LOGIN
Autenticação EPROC
Playwright com SSO Keycloak, TOTP 2FA (autenticação dinâmica), seleção de perfil por OAB. Resiliente a mudanças de sessão e hash de URL.
S2 SCRAPE
Download de Movimentações
Download da planilha XLS de "Últimas Movimentações" com janela de 3 dias. Fonte primária: EPROC direto, sem intermediários.
S3 FILTRAR
Deduplicação e Pré-classificação
Hash SHA-256 para deduplicação. Classificação por regras (74% dos casos sem consumir IA), reduzindo custo por ciclo drasticamente.
S4 NAVEGAR
Acesso ao Processo
Navegação direta ao processo via número CNJ. Leitura da árvore de eventos, partes, documentos e metadados.
S5 LER DOCS
Extração de Documentos
5 camadas de fallback: DOM direto, seletores alternativos, iframe, PDF, OCR. Nenhuma movimentação passa sem documento lido.
S6 ANALISAR
Análise com Claude Sonnet
Classificação A-G: urgência, tipo de atividade, prazo sugerido, ação recomendada. Incluso no custo da API, sem cobrança adicional por IA.
S7 PERSISTIR
Banco Multi-tenant
PostgreSQL com RLS: cada cliente vê apenas seus processos. Supabase com isolamento de tenant por design, não por configuração.
S8 AGIR
Webhooks e Alertas
Webhooks configuráveis por evento, alertas Telegram para urgências, relatório de ciclo com custo e métricas. O cliente recebe, não precisa buscar.
⚡
<24h
Delay de captura
O DataJud, API gratuita do CNJ, tem delay médio de 18 dias. Nossa API captura em horas. 18x mais rápido, na fonte primária.
🧠
A-G

Análise completa do documento com Claude Sonnet: urgência, tipo, prazo, ação sugerida. Nenhum concorrente de API oferece isso.
