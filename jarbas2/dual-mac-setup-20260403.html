FASE 1 — SSH Setup

Fase 1 • ~10 min
No Air (Jarbas faz)
  • ~/.ssh/authorized_keys criado com chmod 600 ✓ pronto
  • Aguardando pub key do Pro ⏳ pendente
No Pro (Christian roda)

Executar os comandos abaixo no Terminal do MacBook Pro:

# 0. Descobrir username real whoami # 1. Verificar/instalar Tailscale which tailscale || brew install tailscale tailscale status # 2. Gerar SSH key dedicada ssh-keygen -t ed25519 -C "macbook-pro" -f ~/.ssh/id_ed25519_air # 3. Mostrar pub key (copiar e me enviar) cat ~/.ssh/id_ed25519_air.pub # 4. Adicionar ao SSH config cat >> ~/.ssh/config << 'EOF' Host air HostName 100.101.208.82 User christian IdentityFile ~/.ssh/id_ed25519_air StrictHostKeyChecking accept-new AddKeysToAgent yes Host air-local HostName 192.168.0.149 User christian IdentityFile ~/.ssh/id_ed25519_air StrictHostKeyChecking accept-new AddKeysToAgent yes EOF # 5. Testar (apos pub key adicionada no Air) ssh air "hostname && whoami && echo SSH_OK"

FASE 2 — Teclado + Mouse Compartilhado

Fase 2 • ~10 min
Opcao A — Universal Control (tentar primeiro)
  • System Settings > General > AirDrop & Handoff > Allow Handoff ON
  • System Settings > Displays > “Link keyboard and mouse to nearby Mac”
  • Se o toggle nao existir: macOS 26 beta pode ter quebrado — ir pro Barrier
Mesma conta iCloud confirmada nos dois Macs — pre-requisito do Universal Control.
Opcao B — Barrier (fallback recomendado)
# Nos DOIS Macs: brew install --cask barrier
  • Pro = Server — compartilha teclado e mouse
  • Air = Client — IP do server: Tailscale IP do Pro
  • Seguro via Tailscale (WireGuard encryption end-to-end)
  • Suporta: mouse, teclado, clipboard compartilhado

FASE 4 — Sync Vaults

Fase 4 • ~10 min
Clone inicial (do Pro)
# Do Pro — clonar vaults: rsync -avz --progress air:~/jarbas-vault/ ~/jarbas-vault/ rsync -avz --progress air:~/mdlab-vault/ ~/mdlab-vault/ rsync -avz --progress air:~/olivia-vault/ ~/olivia-vault/ rsync -avz --progress air:~/cecilia-vault/ ~/cecilia-vault/ rsync -avz --progress air:~/self-improving/ ~/self-improving/
Script de re-sync periodico

Salvar como ~/sync-from-air.sh e rodar quando quiser atualizar:

#!/bin/bash # ~/sync-from-air.sh rsync -avz --delete air:~/jarbas-vault/ ~/jarbas-vault/ rsync -avz --delete air:~/mdlab-vault/ ~/mdlab-vault/ rsync -avz --delete air:~/.claude/agents/ ~/.claude/agents/ rsync -avz --delete air:~/.claude/skills/ ~/.claude/skills/ echo "Sync completo: $(date)"
Usar --delete no re-sync para manter paridade. No clone inicial, nao usar --delete para evitar perder dados locais.
