Cenário n8n Proposto — WH-DOC-01

Workflow principal: Gerar Contratos e Procurações
1. Webhook Trigger
POST /webhook/mdflow/doc/gerar-contratos
Valida header mdflow: MdL1501@
Recebe: lead_id, organizacao_id, tipo, restricao_id, actor_user_id
2. Supabase
Get Lead Data
SELECT * FROM leads WHERE id = lead_id
nome, cpf, rg, cep, cidade, estado, logradouro, numero_endereco, profissao, drive_cliente, codigo_advogado
3. Supabase (paralelo)
Get Restrições
restricoes WHERE lead_id = $1 (empresa)
restricoes_orgao WHERE lead_id = $1 (órgão)
4. Code Node
Prepare Document Queue
Monta array de docs baseado em tipo
Cada item: { template_id, doc_name, variables, folder_id }
5. Loop — SplitInBatches (para cada documento na fila)
5a. Google Docs API
Create from Template
Template ID da fila → preenche variáveis do lead + restrição
5b. HTTP Request
Export as PDF
GET exportLinks.application/pdf do documento criado
5c. Google Drive API
Upload PDF
Upload para pasta do cliente drive_cliente
5d. Google Drive API
Delete Temp Doc
Remove Google Doc temporário (mantém apenas PDF)
6. Google Drive API
Rename Client Folder
Padrão: {codigo_advogado} - {Nome} [{CPF}]
7. Supabase
Update Lead
SET contratos_gerados = true, updated_at = now()
8. Webhook Response
Action-Driven Response
{ success: true, action: "toast_refresh", message: "X documentos gerados", data: { count, folder_url } }

Lógica do Code Node — Document Queue Builder

Nó 4: Prepare Document Queue (pseudocódigo)
const lead = $input.first().json.lead; const restricoes = $input.first().json.restricoes; const restricoesOrgao = $input.first().json.restricoes_orgao; const tipo = $input.first().json.tipo || 'all'; const restricaoId = $input.first().json.restricao_id; const TEMPLATES = { '1.1': { name: 'Contrato Midas', templateId: 'TMPL_1_1', source: 'lead' }, '1.2': { name: 'Termo de Parceria', templateId: 'TMPL_1_2', source: 'lead' }, '1.3': { name: 'Contrato Empresa', templateId: 'TMPL_1_3', source: 'restricao' }, '1.4': { name: 'Termo Ciencia Empresa', templateId: 'TMPL_1_4', source: 'restricao' }, '1.5': { name: 'Termo Veracidade Empresa', templateId: 'TMPL_1_5', source: 'restricao' }, '1.6': { name: 'Contrato Orgao', templateId: 'TMPL_1_6', source: 'restricao_orgao' }, '1.7': { name: 'Termo Ciencia Orgao', templateId: 'TMPL_1_7', source: 'restricao_orgao' }, '1.8': { name: 'Termo Veracidade Orgao', templateId: 'TMPL_1_8', source: 'restricao_orgao' }, '2.1': { name: 'Procuracao Empresa', templateId: 'TMPL_2_1', source: 'restricao' }, '2.2': { name: 'Procuracao Orgao', templateId: 'TMPL_2_2', source: 'restricao_orgao' }, }; const baseVars = { 'NOME COMPLETO': lead.nome, 'CPF': lead.cpf, 'RG': lead.rg, 'CEP': lead.cep, 'CIDADE': lead.cidade, 'ESTADO': lead.estado, 'NOME DA RUA': lead.logradouro, 'NUMERO DO ENDERECO': lead.numero_endereco, 'PROFISSAO': lead.profissao, 'DATA': new Date().toLocaleDateString('pt-BR'), }; const queue = []; // Docs gerais (sem restricao) if (tipo === 'all' || tipo === 'contrato_midas') { queue.push({ ...TEMPLATES['1.1'], variables: baseVars, doc_name: '1.1 - Contrato Midas' }); queue.push({ ...TEMPLATES['1.2'], variables: baseVars, doc_name: '1.2 - Termo de Parceria' }); } // Docs por restricao (empresa) if (tipo === 'all' || tipo === 'contrato_empresa') { for (const r of restricoes) { if (restricaoId && r.id !== restricaoId) continue; const vars = { ...baseVars, VALOR: r.valor, EMPRESA: r.empresa_re, CONTRATO: r.codigo_contrato }; ['1.3', '1.4', '1.5'].forEach(id => { queue.push({ ...TEMPLATES[id], variables: vars, doc_name: `${id} - ${TEMPLATES[id].name} - ${r.empresa_re}` }); }); } } // Procuracoes empresa if (tipo === 'all' || tipo === 'procuracao_empresa') { for (const r of restricoes) { if (restricaoId && r.id !== restricaoId) continue; queue.push({ ...TEMPLATES['2.1'], variables: { ...baseVars, VALOR: r.valor, EMPRESA: r.empresa_re, CONTRATO: r.codigo_contrato }, doc_name: `2.1 - Procuracao - ${r.empresa_re}` }); } } // Docs por restricao (orgao) + procuracoes orgao // Mesmo padrao com restricoesOrgao e templates 1.6, 1.7, 1.8, 2.2 return queue.map(item => ({ json: { ...item, folder_id: lead.drive_cliente } }));
