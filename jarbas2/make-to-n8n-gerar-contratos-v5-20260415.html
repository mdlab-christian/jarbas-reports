5. Os 3 Workflows — Arquitetura

WH-DOC-MIDAS

POST mdflow/doc/gerar-contrato-midas
Input: { lead_id, restricao_id, tipo, oab }
Gera: contrato_midas + termo_parceria = 2 docs
Placeholders: 9-10 (lead data only, sem dados de restrição)

WH-DOC-EMPRESA

POST mdflow/doc/gerar-contrato-honorarios
Input: { lead_id, restricao_id, tipo, oab }
Gera: contrato_empresa + termo_ciencia_empresa + termo_veracidade_empresa + procuracao_empresa = 4 docs
Placeholders: 10-13 (lead + restrição: empresa, contrato, valor)
Nota: {{EMPRESA RESPONSÁVEL}} nos termos ≠ {{EMPRESA}} nos contratos (ambos = restricoes.empresa_nome)

WH-DOC-ORGAO

POST mdflow/doc/gerar-procuracao-orgao
Input: { lead_id, restricao_id, tipo, oab }
Gera: contrato_orgao + termo_ciencia_orgao + termo_veracidade_orgao + procuracao_orgao = 4 docs
Placeholders: 10 (lead data only — nome do órgão está fixo no template)

6. Loop de Geração (Code Node pattern)

Implementação via this.helpers.httpRequest — mesmo padrão do workflow MDFLOW-DRIVE-PASTA-LEAD-v1

// Step 1: Get token const tokenResp = await this.helpers.httpRequest({ method: 'POST', url: 'https://oauth2.googleapis.com/token', headers: { 'Content-Type': 'application/x-www-form-urlencoded' }, body: 'client_id=...&client_secret=...&refresh_token=...&grant_type=refresh_token' }); const token = tokenResp.access_token; // Step 2: Copy template const copy = await this.helpers.httpRequest({ method: 'POST', url: `https://www.googleapis.com/drive/v3/files/${templateId}/copy?fields=id,name`, headers: { 'Authorization': 'Bearer ' + token, 'Content-Type': 'application/json' }, body: JSON.stringify({ name: docName, parents: [folderId] }) }); // Step 3: Replace placeholders await this.helpers.httpRequest({ method: 'POST', url: `https://docs.googleapis.com/v1/documents/${copy.id}:batchUpdate`, headers: { 'Authorization': 'Bearer ' + token, 'Content-Type': 'application/json' }, body: JSON.stringify({ requests: Object.entries(vars).map(([k,v]) => ({ replaceAllText: { containsText: { text: k, matchCase: false }, replaceText: v } }))}) }); // Step 4: Export PDF const pdf = await this.helpers.httpRequest({ method: 'GET', url: `https://www.googleapis.com/drive/v3/files/${copy.id}/export?mimeType=application/pdf`, headers: { 'Authorization': 'Bearer ' + token }, encoding: 'arraybuffer' }); // Step 5: Upload PDF to lead folder const uploaded = await this.helpers.httpRequest({ ... multipart upload ... }); // Step 6: Delete temp Google Doc (the copy, not the template!) await this.helpers.httpRequest({ method: 'DELETE', url: `https://www.googleapis.com/drive/v3/files/${copy.id}`, headers: { 'Authorization': 'Bearer ' + token } });
