Verificar o que você já tem

Rode estes comandos antes de qualquer coisa

Abra o terminal do seu sistema e cole os comandos abaixo.

Terminal (macOS) 
# Verificar Claude Code
claude --version

# Verificar Node.js
node --version

# Verificar Obsidian vault
ls ~/Documents/MdOs-vault/
PowerShell (Windows) 
# Verificar Claude Code
claude --version

# Verificar Node.js
node --version

# Verificar Obsidian vault
ls "$env:USERPROFILE\Documents\MdOs-vault\"

O que fazer com o resultado?

  • ✅ Claude Code apareceu (ex: claude/1.x.x) → pular Etapa 1, ir direto para Etapa 2
  • ❌ "command not found" → fazer Etapa 1 completa
  • Rodei os comandos de verificação no terminal
  • Sei o que já tenho instalado (Claude Code e/ou Node.js)
1

Instalar Claude Code

Só quem recebeu "command not found" na Etapa 0
Se o Claude Code já estava instalado, pule toda esta etapa e vá direto para a Etapa 2.

1A — Instalar Node.js (obrigatório para o Claude Code)

Terminal — Opção A (Homebrew — recomendado) 
brew install node
Opção B — sem Homebrew: baixar o instalador em nodejs.org/pt-br → escolher versão LTS → rodar o instalador.
  1. Abrir: https://nodejs.org/pt-br
  2. Clicar em "Download LTS"
  3. Rodar o instalador .msi → Next, Next, Install
  4. Fechar e reabrir o PowerShell
  5. Confirmar: node --version

1B — Instalar Claude Code

macOS e Windows (PowerShell ou Git Bash) 
npm install -g @anthropic-ai/claude-code

Verificar instalação

Terminal 
claude --version
# Deve aparecer algo como: claude/1.x.x
Windows: se npm não for reconhecido, fechar e reabrir o PowerShell como Administrador.
  • Node.js instalado (node --version mostra a versão)
  • Claude Code instalado (claude --version funciona)
2

Instalar Obsidian e sincronizar o vault

O MdOs usa o Obsidian para sincronizar as skills entre toda a equipe

2A — Instalar o Obsidian

  1. Acessar: https://obsidian.md
  2. Clicar em "Get Obsidian" → baixar para seu sistema
  3. Instalar normalmente

2B — Ativar o Obsidian Sync

  1. Abrir o Obsidian
  2. Ir em Configurações → Sync
  3. Fazer login com a conta que o Christian compartilhou
  4. Selecionar o vault "MdOs-vault"
  5. Aguardar sincronizar (barra de progresso no canto inferior — pode levar 2–5 minutos)

Verificar se sincronizou

Terminal (macOS) 
ls ~/Documents/MdOs-vault/
# Deve aparecer: ACESSOS.md  CLAUDE.md  _tools  install.sh  skills  ...
PowerShell (Windows) 
ls "$env:USERPROFILE\Documents\MdOs-vault\"
# Deve aparecer: ACESSOS.md  CLAUDE.md  _tools  install.sh  skills  ...
Se a pasta aparecer vazia ou não existir: aguardar mais 2 minutos e tentar de novo.
  • Obsidian instalado
  • Obsidian Sync ativado com a conta compartilhada
  • Vault sincronizou (pasta MdOs-vault tem arquivos)

Abrir o MdOs

Fechar e reabrir o terminal é obrigatório
Importante: feche o terminal atual antes de continuar. Isso carrega as variáveis configuradas pelo install.sh.
Terminal (macOS) 
# 1. Fechar o terminal (Cmd+Q ou fechar a janela)
# 2. Abrir um NOVO terminal
# 3. Entrar no vault e abrir o MdOs:
cd ~/Documents/MdOs-vault && claude
Git Bash (Windows) 
# 1. Fechar o Git Bash atual
# 2. Abrir um NOVO Git Bash
# 3. Entrar no vault e abrir o MdOs:
cd ~/Documents/MdOs-vault && claude
  • Fechei o terminal anterior
  • Abri um novo terminal e rodei cd ~/Documents/MdOs-vault && claude
5

Fazer login com sua conta Claude.ai

Obrigatório para voice e limites de uso completos

Dentro do Claude Code (já com o vault aberto), digitar:

Dentro do Claude Code 
/login
  • →Vai abrir o browser automaticamente
  • →Fazer login com sua conta Claude.ai pessoal (ou criar em claude.ai)
  • →Voltar ao terminal
Isso é obrigatório. Sem o login você não consegue usar o voice (segurar ESPAÇO para falar) e os limites de uso são menores.
  • Rodei /login dentro do Claude Code
  • Fiz login no browser com minha conta Claude.ai

Problemas comuns

Soluções rápidas para os erros mais frequentes
"vault não encontrado"
Obsidian não sincronizou ainda. Abrir o Obsidian, aguardar a barra de sync completar, tentar de novo.
"email não encontrado"
Verificar se o email foi digitado exatamente igual à tabela da Etapa 3. Sem espaços, letras minúsculas.
"/mdos:ajuda não reconhecido"
Fechar o terminal, abrir um novo, rodar cd ~/Documents/MdOs-vault && claude de novo.
Claude Code em "API Key mode" (aparece no /status)
Dentro do Claude Code: /login → browser → login com sua conta Claude.ai.
Windows: "bash não reconhecido no PowerShell"
Instalar Git Bash em git-scm.com/downloads e usar ele no lugar do PowerShell.
Mac: "permission denied" ao rodar o install.sh
Rodar chmod +x ~/Documents/MdOs-vault/install.sh e tentar de novo.
