01

O que é o MdOs?

Em uma frase

O MdOs é como um assistente inteligente que sabe tudo sobre os processos do escritório. Você fala (ou digita) o que precisa — e ele busca, analisa e gera o que for necessário, conectando automaticamente ao EPROC, ao banco de dados do escritório e à OlivIA.

Busca

Processos, clientes, intimações, prazos, decisões — tudo instantâneo, sem abrir sistemas separados.

Gera

Petições completas com análise de IA (OlivIA). Fala o número do processo — o DOCX sai em segundos.

Analisa

Intimações do EPROC, urgências do dia, perfil de juízes — classificados e priorizados automaticamente.

Por que usar o MdOs em vez do MdFlow (site)?

⚡
Mais rápido para buscas repetitivas — sem navegar por menus. Um comando e pronto.
🎤
Voice Input — fala em voz alta, o sistema executa. Perfeito para buscas rápidas no meio de uma ligação.
✍️
Petições em segundos — OlivIA integrada. O site tem o mesmo resultado mas o MdOs é mais direto ao ponto.
🤖
Linguagem natural — não precisa decorar nada. "o que tenho para hoje?" funciona.
02

Como abrir o MdOs

Passo único: abrir no terminal

Abra o Terminal (ou iTerm2) e cole este comando:

bash
cd ~/Documents/MdOs-vault && claude

Atalho permanente (configurar uma vez)

Cole este comando no terminal uma única vez. Depois é só digitar mdos para abrir.

bash — executar uma vez
echo "alias mdos='cd ~/Documents/MdOs-vault && claude'" >> ~/.zshrc && source ~/.zshrc

Após configurar o alias, basta digitar mdos em qualquer pasta do terminal. O sistema entra automaticamente no vault correto.

03

Voice Input — o jeito mais fácil

Como usar o Voice Input

1

Com o Claude Code aberto, pressione e segure a barra de ESPAÇO

Um ícone de microfone aparece — o sistema está ouvindo.

2

Fale em português normalmente

Sem pressa. Pode falar o número completo do processo, nome do cliente, o que precisar.

3

Solte o ESPAÇO

O sistema transcreve automaticamente e executa o comando.

Exemplos de falas que funcionam

"quais processos ativos do João?" 🎤 voz
"gera réplica pro processo cinco mil cento e vinte e sete" 🎤 voz
"o que é urgente hoje?" 🎤 voz
"prazos dessa semana" 🎤 voz
"intimações que chegaram hoje" 🎤 voz
"analisa o processo do Bradesco, número cinco mil oitocentos" 🎤 voz

Requisito: estar logado com conta Claude.ai (não API key). Se o Voice não funcionar, digitar /login dentro do Claude Code e fazer login com sua conta pessoal do Claude.

04

Comandos do dia a dia

Clique em qualquer comando para ver exemplos. Todos os exemplos com 🎤 também funcionam por voice input.

/mdos:dia Briefing matinal completo

Mostra tudo que importa para o dia: urgências D-0/D-1, atividades vencendo hoje, petições em andamento, saúde do sistema. Use sempre que chegar no escritório.

/mdos:dia→ briefing completo
"o que tenho para hoje?"🎤
"briefing do dia"🎤
/mdos:urgentes Tudo urgente agora

Agrega intimações classes D/E + atividades vencendo + petições aguardando aprovação. Lista priorizada por urgência.

/mdos:urgentes
"o que é urgente agora?"🎤
"o que tenho que fazer hoje?"🎤
/mdos:status Saúde do sistema

Verifica se OlivIA, Cecília e o banco de dados estão online. Útil quando algum comando não responde.

/mdos:status→ mostra online/offline de cada serviço
/mdos:minha-skills Seus comandos disponíveis

Lista todos os comandos que você tem acesso com base no seu nível (Colaborador/Gestor/Admin).

/mdos:minha-skills
/mdos:processos Buscar processos

Busca processos por advogado, CNJ, banco réu, cliente, ou qualquer combinação. Suporta número parcial.

/mdos:processos ativos do João
/mdos:processos CNJ 5000123-45.2026.8.21.0001
/mdos:processos banco Bradesco
"processos contra o Serasa este mês"🎤
"processos ativos da Heloísa no TJRS"🎤
/mdos:clientes Buscar clientes

Busca clientes por nome, CPF ou contexto processual.

/mdos:clientes João Silva
/mdos:clientes CPF 123.456.789-00
"cliente Maria que tem processo no TJRS"🎤
/mdos:intimacoes Ver intimações recentes

Mostra intimações capturadas pelo Cecília do EPROC, filtradas por data ou urgência.

/mdos:intimacoes de hoje
/mdos:intimacoes urgentes
"intimações que chegaram essa semana"🎤
/mdos:decisoes Consultar decisões judiciais

Consulta decisões no banco. Filtra por resultado, réu, período ou juiz.

/mdos:decisoes improcedentes do mês
/mdos:decisoes favoráveis contra o Bradesco
"decisões de hoje"🎤
/mdos:atividades Ver e criar atividades/tarefas

Consulta atividades e prazos do MdFlow. Pode criar novas atividades por linguagem natural.

/mdos:atividades vencendo hoje
/mdos:atividades do processo 5000123
"criar atividade: prazo contestação processo 5000123 para sexta"🎤

/mdos-olivia:peticao é o comando mais usado no escritório. Gera petições completas em DOCX via IA, diretamente do número do processo.

/mdos-olivia:peticao ⭐ Gerar petição completa com IA

O comando mais usado. Fluxo: identifica o processo → analisa provas → você aprova os tópicos → gera DOCX completo que vai automaticamente para o Google Drive do escritório.

/mdos-olivia:peticao 5000123-45.2026.8.21.0001→ CNJ completo
/mdos-olivia:peticao 5000123→ número parcial funciona
"gera réplica pro processo 5000123"🎤
"gera petição para o caso do João Silva"🎤
/mdos-olivia:analise Analisar processo (sem gerar petição)

Só analisa. Útil quando você quer entender o processo antes de decidir o que fazer. Mostra provas, pontos fortes, pontos fracos, recomendação de tipo de petição.

/mdos-olivia:analise 5000123
"analisa as provas do processo 5000123"🎤
/mdos-olivia:vibelaw Editar petição já gerada

Abre o modo de edição inteligente. Após gerar uma petição, use para reescrever tópicos, adicionar argumentos ou ajustar o tom.

/mdos-olivia:vibelaw
"reescreve o tópico 3 mais formal"🎤
"adiciona jurisprudência do STJ no argumento de dano moral"🎤
/mdos-olivia:lote Gerar petições em lote (Gestor+)

Gera várias petições de uma vez. Requer lista de CNJs num arquivo .txt. Disponível para Gestores e Admin.

/mdos-olivia:lote /tmp/lista-processos.txt
"gera petições para os 5 processos mais urgentes"🎤
/mdos-olivia:juris Buscar jurisprudência

Busca em 1.392+ jurisprudências indexadas do escritório. Retorna as mais relevantes para o caso com percentual de relevância.

/mdos-olivia:juris negativação indevida Serasa
"jurisprudência dano moral banco de dados TJRS"🎤
/mdos-olivia:prazos Prazos por advogado

Lista prazos processuais filtrados por advogado e período.

/mdos-olivia:prazos do João
"prazos da Heloísa essa semana"🎤

Cecília é o robô que acessa o EPROC automaticamente. Ela captura intimações, analisa processos e monitora prazos em tempo real no sistema judicial.

/mdos-cecilia:prazos Prazos críticos da semana

Prazos do EPROC classificados por urgência: D-0 (hoje), D-1 (amanhã), D-3 (próximos 3 dias). Fundo vermelho = ação imediata necessária.

/mdos-cecilia:prazos
"prazos de hoje no EPROC"🎤
"o que vence essa semana?"🎤
/mdos-cecilia:analise Analisar processo no EPROC

Acessa o EPROC, captura movimentações recentes e faz análise completa do processo.

/mdos-cecilia:analise 5000123-45.2026.8.21.0001
"analisa o processo do João Silva no EPROC"🎤
/mdos-cecilia:intimacao Analisar intimação específica

Analisa uma intimação e diz o que fazer: tipo de resposta necessária, prazo, urgência, se precisa gerar petição.

/mdos-cecilia:intimacao [id da intimação]
"o que eu preciso fazer com essa intimação?"🎤
/mdos-cecilia:juiz-perfil Perfil de julgador

Mostra o perfil de um juiz: taxa de sucesso do escritório, padrões de decisão (favorável/neutro/complicador), comarca.

/mdos-cecilia:juiz-perfil Dr. Carlos Souza
"como o juiz Carlos Souza costuma decidir em casos de negativação?"🎤

Controller é o módulo de controle de prazos e produtividade da equipe. Classifica processos por urgência e gera relatórios gerenciais.

/mdos-controller:prazos Prazos D-0, D-1, D-3

Prazos do controller com classificação por urgência. D-0 = hoje, D-1 = amanhã, D-3 = próximos 3 dias.

/mdos-controller:prazos
"prazos de hoje no controller"🎤
/mdos-controller:relatorio Relatório diário (Gestor+)

Gera relatório diário com produtividade da equipe, petições geradas, prazos cumpridos. Disponível para Gestores e Admin.

/mdos-controller:relatorio
"relatório de hoje"🎤
/mdos:juiz Perfil completo de juiz

Consulta o perfil de um juiz de forma isolada, sem precisar estar num processo específico.

/mdos:juiz Dr. Roberto Campos
"como decide o juiz Roberto Campos em negativação?"🎤
/mdos:calculadora Cálculo de valores jurídicos

Calcula valores de causa, dano moral, restrições, juros e correção monetária.

/mdos:calculadora
"calcula valor da causa: dano moral R$ 5.000 + restrição R$ 15.923"🎤
/mdos:dashboard Painel visual do sistema

Gera um HTML visual com membros ativos, skills disponíveis e estado geral do sistema MdOs.

/mdos:dashboard
/mdos:feedback Enviar feedback ao Christian

Envia um feedback ou reporte de problema diretamente para o Christian. Use quando encontrar uma jurisprudência errada, um bug ou tiver sugestão de melhoria.

/mdos:feedback a OlivIA usou jurisprudência errada no processo 5000123
/mdos:feedback o comando de prazos não está mostrando hoje
05

Fluxo de um dia típico

Veja como a equipe usa o MdOs ao longo do dia de trabalho:

8h
08:00 — Chegou no escritório

Briefing matinal: 30 segundos para saber tudo

Sabe tudo: prazos urgentes, intimações novas, petições em andamento, o que a equipe tem para fazer.

/mdos:dia
9h
09:00 — Tem uma intimação urgente

Cecília já analisou. Você só precisa confirmar a ação.

Cecília classifica e diz o que precisa: tipo de resposta, prazo, urgência.

/mdos-cecilia:intimacao [id] "o que eu preciso fazer com essa intimação?"
10h
10:00 — Precisa gerar uma réplica

OlivIA analisa → você aprova os tópicos → DOCX gerado

O DOCX vai automaticamente para o Google Drive do escritório. Tempo médio: 3-4 minutos.

/mdos-olivia:peticao 5000123-45.2026.8.21.0001
14h
14:00 — Cliente ligou perguntando do processo

Histórico completo em segundos

Última movimentação, status atual, documentos — sem precisar abrir o EPROC na mão.

/mdos:clientes João Silva /mdos:processos CNJ 5000123
17h
17:00 — Fechar o dia

Ver o que ficou pendente

Confere se ficou algo urgente sem resolver antes de sair.

/mdos:urgentes "o que ainda não foi feito hoje?"
06

Dicas e truques

💬

Linguagem natural funciona

Não precisa decorar comandos. "quais processos ativos?" funciona igual a /mdos:processos ativos.

🎤

Voice é mais rápido

Segurar ESPAÇO, falar, soltar. Perfeito para buscas rápidas no meio de uma ligação ou reunião.

🔄

Continue a conversa

Após um comando, continue perguntando. "e desse processo, quais são as intimações pendentes?" O sistema mantém o contexto.

⚡

Alias para abrir rápido

Após configurar o alias (seção 2), basta digitar mdos em qualquer pasta do terminal.

🔍

CNJ parcial funciona

/mdos-olivia:peticao 5000123 encontra processos pelo número parcial. Não precisa digitar o CNJ completo.

📁

DOCX vai ao Drive automático

Após gerar uma petição com OlivIA, o arquivo DOCX vai automaticamente para o Google Drive do escritório. Não precisa salvar manualmente.

🌐

MdOs e MdFlow se complementam

Use MdOs para consultas rápidas e geração de petições. Use o MdFlow (site) para visualizações complexas, dashboard e configurações.

📊

Histórico da conversa

O Claude Code mantém o histórico da sessão. Você pode pedir "explica melhor o segundo item" ou "muda o ponto 3" sem repetir contexto.

08

Problemas comuns

Certifique-se de estar dentro do vault correto. No terminal, execute:

cd ~/Documents/MdOs-vault && claude

Se abrir o Claude Code de outra pasta, ele não terá acesso ao contexto do MdOs e não reconhecerá os comandos /mdos:*.

O Voice Input requer login com conta Claude.ai. Dentro do Claude Code, execute:

/login

Siga as instruções para autenticar com sua conta pessoal do Claude.ai. Após o login, o Voice Input estará disponível.

Tente com o número parcial (só os primeiros dígitos). O MdOs faz busca parcial por padrão.

# Em vez de: /mdos:processos CNJ 5000123-45.2026.8.21.0001 # Tente: /mdos:processos 5000123

Se ainda não aparecer, o processo pode não estar no banco do escritório ainda. Verifique no MdFlow (site).

O servidor da OlivIA pode estar reiniciando (acontece algumas vezes por semana). Passos:

  1. Execute /mdos:status para confirmar se OlivIA está offline
  2. Aguarde 2-3 minutos e tente novamente
  3. Se continuar offline, avise o Gustavo no WhatsApp

Verifique sua conexão com a internet. O MdOs precisa de internet para acessar o banco de dados.

  1. Abra qualquer site no navegador para confirmar internet
  2. Execute /mdos:status novamente
  3. Se o banco aparecer como offline mesmo com internet, avise o Gustavo

Cecília captura automaticamente às 6h, 13h e 19h. Se precisar de captura imediata (Gestores+):

/mdos-cecilia:capturar agora

Se mesmo assim não aparecer, pode ser que o EPROC esteja fora do ar (common nos fins de semana). Verifique diretamente no site do EPROC.

09

Suporte

Suporte técnico — Gustavo

Para problemas de sistema, servidor offline, erros que não se resolvem sozinhos. Fale no WhatsApp.

Problemas: OlivIA offline · Banco de dados · EPROC não capturando · Erros de conexão

Melhorias e sugestões — Christian

Para sugestões de novos comandos, melhorias na OlivIA, novos fluxos ou funcionalidades. Use o feedback direto:

/mdos:feedback [sua sugestão aqui]

Dica de ouro: Se não sabe qual comando usar, simplesmente descreva o que precisa em português natural. O MdOs vai entender e executar. "quero ver os processos que vencem essa semana da Heloísa" funciona perfeitamente.
