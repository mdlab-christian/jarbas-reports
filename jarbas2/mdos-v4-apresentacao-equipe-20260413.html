📖 Contexto

O que é o MdOs?

MdOs = MdLab Operating System — o sistema de IA que roda dentro do nosso escritório, conectado a tudo.

💻 Como usar
Abra um terminal no seu computador, entre na pasta MdOs-vault e converse com o Claude Code. Os comandos ficam disponíveis como /mdos-olivia:peticao.
🔄 Sincronização automática
Quando o Christian atualiza uma skill ou adiciona um novo comando, o Obsidian Sync distribui automaticamente para todos os computadores da equipe. Sem instalar nada.
🏗️ Estrutura
Seus comandos vivem no Mac Mini do escritório + Supabase. A IA processa, você executa. Tudo controlado, auditável, seguro.
MacBook da equipe → cd ~/Documents/MdOs-vault && claude │ ├── /mdos-olivia:peticao → Mac Mini → Claude API → DOCX ├── /mdos-cecilia:prazos → Mac Mini → EPROC → Supabase ├── /mdos-processos:consultar → Supabase (5.800 processos) └── /mdos-mia:briefing → Mac Mini → Telegram @mia_mdbot

🔧 Infraestrutura

Como a MIA funciona

Simples de usar, sofisticada por dentro. Roda no mesmo Mac Mini que hospeda OlivIA e Cecilia.

Você └── Telegram: @mia_mdbot │ ▼ MIA Daemon (Mac Mini :18795) ├── Lê: SOUL.md (quem ela é) ├── Lê: USER.md (contexto escritório) ├── Consulta: Supabase → 5.800 proc ├── Verifica: OlivIA :19100 └── Responde: Telegram ← você
🖥️ Onde roda
Mac Mini M4 do escritório. Usuário dedicado mia (UID 508), porta 18795. LaunchAgent garante reinício automático.
🔒 Segurança
DM Policy = allowlist. Só membros autorizados recebem resposta. Tokens em arquivo .env com permissão 600. Nunca hardcoded.
🧠 Modelo IA
Claude Sonnet para conversas (rápido e preciso). Claude Haiku para crons automáticos (econômico). Split inteligente de custo.
