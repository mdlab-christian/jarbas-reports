Sim, e possivel criar templates proprios — nao e necessario usar templates pre-definidos da Meta.

Passo a passo:

  1. Acessar Meta Business Manager → WhatsApp → Templates de Mensagem
  2. Clicar em Criar novo
  3. Preencher nome, categoria (MARKETING), idioma (pt_BR) e corpo da mensagem
  4. Submeter para analise
  5. Aguardar aprovacao: ⏳ 24-48h
  6. Apos aprovado: usar via Cloud API com o nome do template
