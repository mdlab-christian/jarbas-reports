Problema3x chamadas Claude extras quando revisor roda = +3-5 min de latência
ImpactoBloqueia geração quando score falha ou API lenta — UX inaceitável
AçãoModo padrão = sem revisor. Modo avançado = revisor max 2 tentativas

REMOVER T149 — Classificação Manual de Provas no Upload
ProblemaCombobox de tipo de documento no upload — Mac Mini ignora o valor

Decisão: remoção limpa — simplifica upload e reduz erros do usuário
OTIMIZAR T174 — S7 Docs Timeout
ProblemaDOCX/PDF bloqueiam status CONCLUIDO quando geração demora
ImpactoUsuário aguarda Step 4 enquanto docs ainda processam em background
AçãoPromise.allSettled + 30s timeout + background delivery via Realtime
Decisão: CONCLUIDO não espera S7 — docs entregues via Realtime separado
REMOVER T164 — Campos Inúteis S1
Problema14 campos buscados no banco em S1 mas nunca usados na montagem do prompt
ImpactoQueries mais lentas, payload maior, latência de S1 sem necessidade

REMOVER T175 — S5 Fallback Markdown
ProblemaS5 converte Markdown para HTML quando LLM não segue instrução
ImpactoCódigo morto que mascara falhas do prompt — dificulta debugging
AçãoRemover fallback. Reforçar instrução no prompt para HTML direto
Decisão: remover — forçar disciplina no S4 em vez de compensar em S5
