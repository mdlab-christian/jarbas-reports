Step 2 — Analise de Provas
EF olivia-topicos dispara → INSERT olivia_historico (pct=1, status=PROCESSANDO) → Mac Mini poll 5s detecta → Pipeline ANALISE

Carrega: SOUL.md + STYLE.md + playbooks + restricoesNOVO + memoria empresaNOVO
Claude Sonnet/Opus analisa provas → Output: topicos + tag_answers + perguntasNOVO

Frontend Realtime detecta TOPICOS_PRONTOS → Usuario responde perguntasNOVO, reordena topicos, aprova
Step 3 — Geracao
EF olivia-gerar dispara → Mac Mini detecta → Pipeline GERACAO

detectarComplexidade()NOVO: simples vs complexo
Carrega: modelo_skeleton + juris via Smart Juris v2NOVO + memoria empresaNOVO
Claude Opus 4.6 SEMPRE gera peticao → formatarHTML()NOVO aplica formatacao
substituirCamposDados() preenche placeholders

TipTap editor carrega HTML → VibeLaw slash commands para refinamento → Autosave 30s
Step 4 — Conclusao
Gera DOCX + PDF (WeasyPrintNOVO) → Relatorio (pontos especificosNOVO)
