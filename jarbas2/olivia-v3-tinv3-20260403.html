olivia-smartblocks.mjs — Fluxo de Montagem
⚠️ Não existe ceiling de 8K chars explícito

O estimarTokens() na linha 591 é apenas logging. Não há validação, não há hard-stop. O suposto limite de 8K era informal.

Fluxo de Montagem (6 passos)
  1. buscarDadosProcesso(env, processoId) — 4 queries Supabase: processo + cliente + advogado + comarca
  2. Infere gênero do cliente (terminações femininas no nome)
  3. Formata dados dinâmicos: qualificacao_sexo, dataHoje, oabFormatado, cidade
  4. substituirVarsDinamicas(html, dados) — substitui {{campo}} no template HTML
  5. buscarTagsDoGrupo(env, mgId) — olivia_modelos_grupo → active_version_id → olivia_modelo_tags JOIN olivia_tags
  6. montarHtmlComBlocos() — prefere conteudo_template_html_rico (mammoth, bold/italic nativos) se disponível; sanitizarBlocoHtml() remove instruções internas CAPS
