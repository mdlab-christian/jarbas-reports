🔷 Edge Function olivia-topicos (Deno)
// INPUT
{
  processo_id,
  tipo_peticao_id,
  modelo_id,     // olivia_modelos_grupo.id
  versao_id,     // olivia_modelos_peticao.id (opcional)
  advogado_id,
  organizacao_id,
  qualidade: "padrao" | "avancado",
  doc_urls: string[]  // URLs das provas enviadas
}

// AÇÕES DA EF
1. Valida JWT Bearer
2. Cria row em olivia_historico (status: PENDENTE)
3. POST Mac Mini /olivia/pipeline-v3?modo=topicos
   → NÃO aguarda resposta (202 ack apenas)
4. Retorna { historico_id } imediatamente

// OUTPUT
{ success: true, historico_id: "uuid", status: "PENDENTE" }

// INPUT
{
  historico_id: string,

    { id, titulo, ordem, instrucoes? }
  ],

// AÇÕES DA EF
1. Valida JWT + busca historico
2. Atualiza olivia_topicos:
   - titulo, ordem, consideracoes (instrucoes)
   - soft-delete não aprovados (deleted_at = now())
3. Resolve versao_id:
   grupo_id → olivia_modelos_grupo.active_version_id
4. POST Mac Mini /olivia/pipeline-v3?modo=gerar
   → NÃO aguarda (202 ack apenas)
5. Retorna { success: true } imediatamente

// OUTPUT
{ success: true }

⌨️ Slash Commands (Cmd+K)
ComandoO que faz
/reescreverReescreve seleção — linguagem jurídica precisa, sem alterar argumentos
/expandirAdiciona fundamentos legais (CDC, STJ, LGPD) ao trecho
/simplificarTorna o texto mais claro sem perder precisão
/formalizarEleva registro para linguagem forense formal
/jurisprudenciaInsere jurisprudência relevante (futuro)
🔷 Edge Function olivia-vibelaw (slash commands)
// INPUT (Step 3 — VibeLaw)
{
  comando: "reescrever"|"expandir"|"simplificar"|"formalizar",
  trecho: string,    // max 8.000 chars
  contexto?: string  // contexto extra do processo
}

// OUTPUT
{
  novo_trecho: string,
  diff_patches: DiffPatch[]  // verde/vermelho no DiffHighlighter
}

// Modelo: Claude Sonnet 4.6 · Timeout: 30s
🔄 Fluxo do Autosave
1. Editor muda → useVibeLaw marca dirty
2. Timer 30s → supabase.from('olivia_historico').update({ peticao_html })
3. Split view: EditorPreviewPane mostra preview HTML em tempo real (D13)
4. SectionNavV3 (minimap TOC lateral) atualiza ao detectar H1/H2 no doc (D13)
5. DiffHighlighter: mudanças via VibeLaw ficam em verde/vermelho (D13)
Estado inicial: se peticao_html ainda null quando o Step 3 abre, mostra spinner "Gerando petição..." e aguarda Realtime entregar o HTML.
💾 Clica "Finalizar" → salva versão final do peticao_html → vai para Step 4
