4 Limitação Crítica da API ADVBOX
🔴
A API ADVBOX não tem endpoint para ATUALIZAR customers de um processo existente
O PUT /lawsuits/{id} aceita apenas: stages_id, type_lawsuits_id, process_number, protocol_number, folder, date, notes. Não aceita customers[]. Isso significa que corrigir a parte autora ou empresa ré na ADVBOX via API pode não ser possível diretamente. Precisamos investigar antes de executar.
