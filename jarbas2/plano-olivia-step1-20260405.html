⚠️ Problemas Identificados
🔴
Busca de processos quebrada (P0)
SearchBoxProcesso chama useOliviaProcessoBusca(search, orgId) mas o hook atual tem assinatura (query) sem orgId. Campos também divergem: numero_cnj vs numero_processo, clientes.nome vs cliente_nome, etc.
🟡
Labels errados em QualidadeIaSelector
Mostra "Sonnet 4.6" e "Opus 4.6" — deveria mostrar "Normal" e "Avançado" conforme spec.
🟡
Toggle "Análise do Processo" não existe
Spec exige toggle Sim/Não que determina se a IA faz análise completa de provas ou apenas gera petição com dados do backend. Campo analisarProcesso não existe no store Zustand.
🟡
Upload de documentos mal posicionado e sem condicional
O componente UploadDocumentosV3 aparece sempre, mas deveria aparecer somente quando análise = Sim, logo abaixo do toggle. Usa Drawer lateral — spec pede Modal central (Dialog).
🔵
Modal de documentos sem classificação por doc
Spec exige combobox de classificação por documento (Contrato, Contestação, Extrato, Sentença, etc.) e botão "Analisar Processo Completo" (integração EPROC futura).
🔵
analisarProcesso não propagado para Edge Function
Quando análise = não, olivia-topicos deve saber para não gerar relatório HTML e usar apenas dados do backend + modelo.
