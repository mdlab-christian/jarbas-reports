Comandos para o MacBook Pro

Copie e cole cada comando separadamente no Terminal. Espere cada um terminar antes de rodar o proximo.

1
Apagar o SSH config corrompido
Remove o arquivo que ficou com lixo.
rm -f ~/.ssh/config
2
Criar SSH config novo para o Air (IP local)
Conecta via rede local 192.168.0.149. Depois adicionamos Tailscale.
printf "Host air\n  HostName 192.168.0.149\n  User christian\n  IdentityFile ~/.ssh/id_ed25519_air\n  StrictHostKeyChecking accept-new\n  AddKeysToAgent yes\n" > ~/.ssh/config
3
Corrigir permissao do config
SSH exige permissao 600 no config.
chmod 600 ~/.ssh/config
4
Verificar que o config ficou correto
Deve mostrar Host air com HostName 192.168.0.149.
cat ~/.ssh/config
5
Testar conexao SSH com o Air
Se pedir "yes/no" para fingerprint, digite yes. Deve retornar o hostname do Air e "SSH_OK".
ssh air "hostname && whoami && echo SSH_OK"
Resultado esperado: Macbook-air---Jarbas-OS.local / christian / SSH_OK
6
Iniciar Tailscale
Vai abrir uma URL no browser para autenticar. Faca login com sua conta.
brew services start tailscale
7
Autenticar no Tailscale
Vai mostrar uma URL. Abra no browser e autorize.
tailscale up
8
Verificar Tailscale conectado
Deve mostrar o MacBook Air (jarbas-macbookair) na lista.
tailscale status
9
Adicionar host Tailscale ao SSH config
Adiciona entrada "air-tail" para conexao via Tailscale (funciona de qualquer rede).
printf "\nHost air-tail\n  HostName 100.101.208.82\n  User christian\n  IdentityFile ~/.ssh/id_ed25519_air\n  StrictHostKeyChecking accept-new\n  AddKeysToAgent yes\n" >> ~/.ssh/config
10
Testar SSH via Tailscale
Mesma coisa do passo 5, mas via VPN Tailscale.
ssh air-tail "hostname && whoami && echo TAILSCALE_OK"
Resultado esperado: Macbook-air---Jarbas-OS.local / christian / TAILSCALE_OK
