Fluxo Real de Uso

# Sessao 1: Spec (absorve contexto automatico)

→ Le CLAUDE.md: React+TS+Supabase+n8n, DS v5, 12 principios

→ Verifica schema Supabase: tabela audiencias nao existe
→ Gera .projeto/spec.md com convencoes absorvidas
→ "/clear + /projeto break"

# Sessao 2: Break por camadas
/projeto break
→ Backend: T01 migration (S), T02 RPC (S)
→ Frontend: T03 pagina layout (M), T04 modal criar (M)
→ n8n: T05 webhook notificacao (M)
→ Polish: T06 loading/empty + responsivo (S)
→ "/clear + /projeto next"

# Sessao 3+: Loop com /projeto next
/projeto next
→ Proxima: T01 (deps OK, backend, S)
→ Plan: encontrou pattern em processos, skill: /mdflow-migration
→ Execute: migration aplicada via Supabase MCP
→ Validate: schema OK, RLS OK
→ Commit: feat(audiencias): cria tabela audiencias com RLS
→ Push: origin main
→ T01 validada (1/6 = 16%). /clear + /projeto next
