🧠 Lições Aprendidas Hoje
🚨 Crítico — auth-profiles.json

Ao trocar chave de agente OpenClaw, sempre atualizar DOIS arquivos:

1. ~/.env (ou /Users/[agente]/.env)
2. ~/.openclaw/agents/main/agent/auth-profiles.json

O auth-profiles.json tem precedência sobre o .env — a chave velha fica em cache e gera 401 mesmo com .env correto.

⚠️ Atenção — Whisper CLI vs API

Whisper CLI pode travar a 185% CPU processando arquivos grandes. Configurar OpenAI Whisper API como primário e CLI como fallback:

provider: openai · model: whisper-1
~2s por arquivo · confiável

Verificar se OPENAI_API_KEY está no LaunchDaemon plist (não só no .env do usuário).

📋 ADVBOX — Armadilhas Críticas
  • → POST /movements usa DD/MM/YYYY (único endpoint diferente)
  • → lawsuits_id em transactions exige customers_id junto
  • → amount=0 causa erro 500
  • → postalcode DEVE ter hífen (99999-999)
  • → date_payment não pode ser futura
  • → GET /movements sem resultado → 204 (não 404)
