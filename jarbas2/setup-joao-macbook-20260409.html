Fase 1 — Ativar Acesso Remoto (SSH)

1
Permitir que o Christian acesse teu Mac remotamente
💡 Isso permite que o Christian configure coisas no teu Mac sem precisar estar do lado. Funciona apenas na mesma rede Wi-Fi.
Abrir Ajustes do Sistema
Clica no icone da maca () no canto superior esquerdo da tela e depois em "Ajustes do Sistema"
 (maca) <-- canto superior esquerdo da tela ├─ Sobre Este Mac ├─ Ajustes do Sistema... <-- clica aqui ├─ App Store... └─ ...
Ir em Geral e depois Compartilhamento
Na barra lateral esquerda dos Ajustes, clica em "Geral" e depois em "Compartilhamento"
Ajustes do Sistema ├─ Wi-Fi ├─ Bluetooth ├─ Geral <-- clica aqui primeiro │ ├─ Sobre │ ├─ Armazenamento │ ├─ Compartilhamento <-- depois aqui │ └─ ...
Ativar "Login Remoto" (Remote Login)
Procura a opcao "Login Remoto" e ativa o botao ao lado. Pode pedir tua senha do Mac — digita e confirma.
Compartilhamento ├─ Compartilhamento de Tela ............. ○ ├─ Login Remoto .......................... ◉ <-- ATIVAR ├─ Compartilhamento de Arquivos ......... ○ └─ ...
⚠️ Depois de ativar, vai aparecer algo tipo: ssh joao@joao-MacBook-Air.local — Esse texto e importante! Anota ele.
Anotar o comando SSH e mandar pro Christian
Copia o texto que apareceu (tipo ssh joao@joao-macbook.local) e manda pro Christian no WhatsApp.
✅ Fase 1 concluida! O Christian agora pode acessar teu Mac pela rede.
Marcar Fase 1 como concluida
🤖

Fase 2 — Instalar Claude Code

2
Instalar as ferramentas no teu Mac
💡 Agora vamos usar o Terminal. E uma tela preta onde tu digita comandos. Nao se preocupa — e so copiar e colar!
Abrir o Terminal
Aperta Cmd + Espaco (barra de espaco) ao mesmo tempo. Vai aparecer uma barra de busca. Digita "Terminal" e aperta Enter.
┌─────────────────────────────────────────┐ │ 🔍 Terminal │ └─────────────────────────────────────────┘ ↑ Spotlight (Cmd + Espaco) Resultado: ├─ Terminal <-- aperta Enter └─ ...
💻 O Terminal e aquela janela preta/branca com texto. E normal parecer confuso — tu so precisa copiar e colar os comandos!
Instalar o Homebrew (gerenciador de pacotes)
Copia o comando abaixo e cola no Terminal (Cmd+V). Depois aperta Enter.
Cola no Terminal e aperta Enter
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
⚠️ Vai pedir tua senha do Mac. Quando digitar, nao vai aparecer nada na tela (nem bolinhas) — e normal, o Mac faz isso por seguranca. Digita a senha e aperta Enter.
⏳ Demora uns 2-5 minutos. Espera ate aparecer uma mensagem de sucesso. Nao fecha o Terminal!
🚨 IMPORTANTE: No final, o Homebrew mostra "Next steps" com 2 ou 3 comandos para rodar. Tu PRECISA copiar e colar esses comandos tambem! Eles adicionam o Homebrew ao sistema. Se pular isso, os proximos passos nao vao funcionar.
👁️ Os comandos do "Next steps" geralmente sao algo tipo:
echo >> ~/.zprofile
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
Copia e cola cada um deles no Terminal, um por um.
Instalar o Node.js
Copia e cola no Terminal:
brew install node
⏳ Espera terminar. Pode demorar 1-2 minutos.
Instalar o Claude Code
Copia e cola no Terminal:
npm install -g @anthropic-ai/claude-code
Testar se deu certo
Copia e cola no Terminal:
claude --version
✅ Se aparecer um numero de versao (tipo 1.0.12), funcionou! Se deu erro, veja a secao "Se Deu Erro" la embaixo.
Marcar Fase 2 como concluida
📨

Fase 3 — Mandar Info pro Christian

3
Enviar dados do teu Mac
💡 O Christian precisa dessas infos pra acessar teu Mac. Roda os 2 comandos e manda o resultado no WhatsApp.
Descobrir teu nome de usuario
Copia e cola no Terminal:
whoami
Vai aparecer algo tipo joao — anota isso.
Descobrir o nome do teu Mac
Copia e cola no Terminal:
hostname
Vai aparecer algo tipo Joaos-MacBook-Air.local — anota isso tambem.
Mandar tudo pro Christian no WhatsApp
Manda uma mensagem pro Christian com essas infos:
Mensagem pro Christian: "Fala Christian! Fiz o setup do Mac. Meu usuario e: [resultado do whoami] Meu hostname e: [resultado do hostname] SSH ta ativado e Claude Code instalado. 👍"
🎉 Pronto! Tu terminou tudo. Agora o Christian vai configurar o resto remotamente.
Marcar Fase 3 como concluida

Se Deu Erro

🔧 Problemas comuns e como resolver
❓ "Command not found: brew" depois de instalar o Homebrew
Tu provavelmente pulou os comandos do "Next steps" que apareceram no final da instalacao. Fecha o Terminal, abre de novo, e roda:
eval "$(/opt/homebrew/bin/brew shellenv)"
❓ "Command not found: node" depois de instalar
Fecha o Terminal e abre de novo. Se continuar, roda o comando acima primeiro e depois tenta brew install node de novo.
❓ "Command not found: npm"
O npm vem junto com o Node.js. Se nao funcionou, o Node nao instalou direito. Roda de novo: brew install node
❓ "Permission denied" ou "EACCES"
Roda o comando com sudo na frente. Exemplo:
sudo npm install -g @anthropic-ai/claude-code
Vai pedir tua senha do Mac de novo (lembra: nao aparece nada na tela enquanto digita).
❓ "Password:" aparece e nao sei a senha
E a mesma senha que tu usa pra desbloquear o Mac (quando liga ou quando volta do descanso de tela). Se nao lembra, manda mensagem pro Christian.
❓ Nao acho o "Login Remoto" nos Ajustes
No macOS em ingles e "Remote Login". Se teu Mac estiver em ingles, procura por: System Settings > General > Sharing > Remote Login
❓ Deu qualquer outro erro estranho
Tira um print da tela (Cmd+Shift+3) e manda pro Christian no WhatsApp. Ele resolve!
