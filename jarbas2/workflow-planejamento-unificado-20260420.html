# Sessão de Planejamento — {PROJETO} — {DATA} ## Meta da Sessão O que Christian quer resolver/decidir nesta sessão (1-3 frases). ## Contexto Discutido - Estado atual do projeto: W{N} em execução, {X}/{Y} tasks - Bloqueadores identificados: T0X (desc), T0Y (desc) - Decisões pendentes do CTO: D-XX (desc) - Contexto adicional relevante: ... ## Decisões Tomadas - [ ] D-01: {decisão} — {motivo} - [ ] D-02: {decisão} — {motivo} ## Artefatos Gerados nesta Sessão - [ ] {arquivo/spec gerado aqui} → salvar em {path} - [ ] {outro artefato} ## Prompts Prontos para Claude Code ### Prompt 1 — {contexto} ``` {prompt exato, copiável, sem ambiguidade} ``` Arquivos afetados: {lista} Tempo estimado: {Xmin} ### Prompt 2 — {contexto} ``` {prompt exato} ``` ## Ordem de Execução 1. Prompt 1 → gate: {como validar que funcionou} 2. Prompt 2 → gate: {como validar} ## O que NÃO fazer nesta execução - [ ] {anti-pattern específico} ## Próximos Passos (pós-execução CC) - {o que revisar manualmente} - {próxima sessão Claude.ai: tema sugerido} --- _Gerado: {DATA} {HORA} | Jarbas 2.0 Claude.ai_ _Executado: [pendente] | Claude Code {Air|Pro}_

Skill /projeto:plan-session — Lógica de Execução

Pseudocódigo da skill (a ser implementada em T-03)
# /projeto:plan-session [nome-projeto] # FASE 1: Localizar sessão mais recente VAULT=~/jarbas-vault/02_projetos/{nome}/sessoes/ SESSAO=$(ls -t $VAULT/*.md | head -1) # Se não encontrar: erro amigável + instrução # FASE 2: Ler e apresentar resumo cat $SESSAO → extrair: - Meta da sessão - Decisões pendentes ([ ]) - Quantidade de prompts prontos - Estimativa total de tempo # FASE 3: Confirmação do CTO echo "Sessão: $(basename $SESSAO)" echo "Decisões: X | Prompts: Y | Tempo est.: Zmin" echo "Executar? [s/n/ver-prompts]" # FASE 4: Execução em automode for prompt in prompts_da_sessao: - Exibir contexto do prompt - Executar em automode com limite de tokens - Validar gate definido - Marcar decisão como [x] na sessão - Registrar log em .projeto/{nome}/logs/ # FASE 5: Update vault pós-execução STATUS_ATUAL=~/jarbas-vault/02_projetos/{nome}/STATUS-ATUAL.md cat > $STATUS_ATUAL: - Data da última execução CC - Tasks concluídas / total - Próximos bloqueadores - Sugestão para próxima sessão Claude.ai # FASE 6: Update CONTEXTO.md global sed -i "s/Última sessão .*/Última sessão: $(date)/" ~/jarbas-vault/00_cerebro/CONTEXTO.md
