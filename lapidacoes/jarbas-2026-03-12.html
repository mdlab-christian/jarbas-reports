Problemas Identificados & Corrigidos

P0useJarbasHeartbeat — coluna errada para online status
O hook usava updated_at para calcular quando o JARBAS foi visto, mas a coluna correta é last_seen. Interface do banco incompleta: faltavam instance_id, uptime_seconds, versao, metadata, movimentacoes_hoje. Mission Control mostrava status e ping incorretos.
- .order('updated_at', { ascending: false }) + .order('last_seen', { ascending: false }) - const diffMin = hb ? (agora - new Date(hb.updated_at).getTime()) / 60000 : null + const diffMin = hb ? (agora - new Date(hb.last_seen).getTime()) / 60000 : null
P1conversas_section — cores Tailwind named (DS drift)
Avatar do JARBAS usava bg-green-500/20 e text-green-400 (anti-padrão DS v5).
- bg-green-500/20 | text-green-400 + bg-primary/15 | text-primary
P1jarbas_command_palette — 5 abas fantasma no ⌘K
⌘K listava cockpit, operacao, estrategia, knowledge, mdlab — abas que não existem em JarbasTab. Clicar silenciosamente navegava para o default. Corrigido: apenas as 7 abas reais.
P1jarbas_shell — import morto ChatSection
ChatSection importado mas nunca renderizado (switch 'chat' usa ConversasSection). Import removido.
DSterminal_live — comentário de exceção para hex hardcoded
xterm Terminal.theme usa Canvas API (não suporta CSS variables). Adicionado comentário explícito de exceção DS (padrão PATTERNS.md §confetti).
