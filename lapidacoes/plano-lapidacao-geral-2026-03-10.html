💬 Grupos Telegram — Um por Página P0+P1

Crie um grupo para cada página P0 e P1. Adicione o bot. O JARBAS notifica cada lapidação no grupo da página correspondente.

Convenção de nome: MdFlow — /controller, MdFlow — /olivia, etc.
Depois de criar: Copie o chat_id e atualize ~/jarbas/config/page-groups.json
Para pegar o chat_id: Adicione o bot ao grupo e mande /autorizar
page-groups.json
{ "pages": {
  "/controller": null, ← preencher com chat_id
  "/olivia": null,
  "/acordos": null,
  ... demais páginas ...
} }

⚠️ Grupos não configurados → notificações vão para o grupo padrão (-5251265489). Funciona mesmo sem grupos por página.

⚡ Comandos — Como Ativar e Controlar

▶️ Ativar — inicia tudo em Turbo Mode

/fila-ativar

Inicializa missão completa (38 páginas), Turbo Mode ON, watchdog ativo. RAM decide quantas rodam em paralelo.

⏸️ Desativar — pausa sem matar execução atual

/fila-desativar

A página em execução termina normalmente. Nenhuma nova página começa até você reativar.

➕ Adicionar páginas específicas à fila

/fila-adicionar /acordos /decisoes

Adiciona páginas avulsas sem iniciar a missão completa. Útil para reprocessar uma página específica.

📊 Acompanhar

status
/fila

status = RAM + páginas em execução + ETA. /fila = lista detalhada da fila.

✅ Checklist — Iniciar Agora

1

Fechar Chrome e apps pesados

Libera ~4GB RAM. Meta: RAM < 45% antes de iniciar → 3 páginas paralelas.

2

Subir dev server (opcional, para screenshots)

cd ~/beta-mdflow && npm run dev -- --host
Se não subir, screenshots são puladas. Lapidação continua normal.

3

Disparar lapidação geral

Envie no Telegram: /fila-ativar
JARBAS confirma, inicia missão e ativa watchdog automaticamente.

4

Criar grupos por página (pode fazer enquanto roda)

Crie grupos no Telegram, adicione o bot, mande /autorizar em cada um.
Preencha ~/jarbas/config/page-groups.json com os chat_ids.
JARBAS começa a usar automaticamente nas próximas páginas.

5

Dormir e acompanhar ao acordar

Use status pela manhã. Verifique relatórios nos grupos de cada página.
Páginas com problema: retry /pagina ou pular /pagina.
