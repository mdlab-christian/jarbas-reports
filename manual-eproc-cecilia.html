1. Arquitetura do EPROC (iframes)

O EPROC usa uma estrutura de dois frames aninhados. O page.evaluate() padrão só enxerga o documento raiz — que é basicamente um container vazio. Todo o conteúdo real fica nos frames internos.

// Estrutura real do DOM no EPROC:
window (externo_controlador.php)
  └── <frameset> ou <iframe id="ifrContent">
        ├── frame#topo  (menu lateral)
        └── frame#conteudo  ← conteúdo real aqui
              └── <table>, <form>, processos...
⚠️
Regra #1: Nunca use page.evaluate() direto para extrair dados. Sempre itere page.frames() e procure o frame que tem o conteúdo.
// ✅ CORRETO — iterar frames
const frames = page.frames();
for (const frame of frames) {
  const text = await frame.evaluate(() => document.body?.innerText || '');
  if (text.includes('CNJ') || text.match(/\d{7}-\d{2}/)) {
    // Este é o frame certo
    const cnjs = text.match(/\d{7}-\d{2}\.\d{4}\.\d\.\d{2}\.\d{4}/g) || [];
    console.log(`Frame certo: ${cnjs.length} CNJs`);
    break;
  }
}

// ❌ ERRADO — só vê o container vazio
const texto = await page.evaluate(() => document.body.innerText);
// → retorna "" ou "0 tables"

3. Fluxo de login (state machine)

O login tem até 5 estados diferentes dependendo do tribunal. A máquina de estados do JARBAS trata cada um:

EstadoOnde apareceAção
EPROC_LOGIN Tela inicial TJSP/TJRS antigo Preencher #username + #password, clicar Entrar
SSO_LOGIN Keycloak (TJRS/TJSC novos) Preencher com [name=password] (não #password — pode estar hidden)
2FA Após senha, pede TOTP Gerar código TOTP + marcar checkbox "Lembrar dispositivo"
PROFILE SC074025: mostra RS116571 e SC074025 Clicar no botão acaoLogar que contém o OAB correto no texto
PANEL Painel do advogado — login OK Navegar para URL de lista
⚠️
SC074025 cai na seleção de perfil porque a conta tem dois perfis (RS116571 e SC074025). O seletor button[onclick*="acaoLogar"] lista os botões — escolher o que contém "SC074025" no texto.
// Seleção de perfil — procurar pelo OAB no texto do botão
const oab = 'SC074025';
await page.evaluate((oab) => {
  const btns = [...document.querySelectorAll('button[onclick*="acaoLogar"]')];
  const btn = btns.find(b => b.textContent.toUpperCase().includes(oab));
  if (btn) btn.click();
}, oab);
await page.waitForLoadState('domcontentloaded', { timeout: 30000 });

4. Navegar para lista de processos

🚨
Não use page.goto() para navegar dentro do EPROC. O EPROC mantém sessão via cookies que são destruídos se o goto disparar um redirect. Use sempre page.evaluate(() => window.location.href = url) ou link.click().
  1. Após login, extrair a base URL da sessão atual 
    const currentUrl = page.url();
const baseUrl = currentUrl.match(/^https?:\/\/[^/]+/)?.[0] || '';
// ex: "https://eproc1g.tjsc.jus.br"
  2. Montar a URL de lista e navegar via evaluate 
    const listUrl = `${baseUrl}/eproc/controlador.php?acao=relatorio_processo_procurador_listar&ord_ultimas_movimentacoes=S`;
await page.evaluate((url) => { window.location.href = url; }, listUrl);
await page.waitForLoadState('domcontentloaded', { timeout: 30000 });
await page.waitForTimeout(2000); // aguardar frames carregarem
  3. Verificar se chegou na lista (não num redirect) 
    const finalUrl = page.url();
if (finalUrl.includes('externo_controlador') || finalUrl.includes('painel')) {
  // Sessão presa — fazer o goto de novo ou checar login
  console.log('⚠️ Redirecionado para painel — sessão ou URL incorreta');
}
  4. RS116571: página em branco? Checar iframe não renderizado

    O RS116571 cai no painel e o conteúdo fica num iframe não visível. Aguardar networkidle antes de extrair.

    await page.waitForLoadState('networkidle', { timeout: 20000 }).catch(() => {});
await page.waitForTimeout(1500);

5. Extrair dados do iframe correto

Após navegar para a lista, o conteúdo fica num frame interno. O page.evaluate() direto retorna 0 tables porque só vê o documento raiz (shell vazio).

// ─── Função auxiliar: encontrar frame com conteúdo real ───────────────
async function getContentFrame(page) {
  const frames = page.frames();
  for (const frame of frames) {
    try {
      const text = await frame.evaluate(() => document.body?.innerText || '');
      // Frame de conteúdo tem processos OU tem tabela
      const hasCnj = text.match(/\d{7}-\d{2}\.\d{4}/);
      const hasTable = await frame.evaluate(() =>
        document.querySelectorAll('table tr').length > 2
      );
      if (hasCnj || hasTable) return frame;
    } catch { /* frame cross-origin ou vazio */ }
  }
  return null;
}

// ─── Uso ───────────────────────────────────────────────────────────────
const contentFrame = await getContentFrame(page);
if (!contentFrame) {
  console.log('❌ Nenhum frame com conteúdo encontrado');
  // Tentar aguardar mais e repetir
}

// Extrair CNJs do frame certo
const cnjs = await contentFrame.evaluate(() => {
  const regex = /\d{7}-\d{2}\.\d{4}\.\d\.\d{2}\.\d{4}/g;
  const text = document.body?.innerText || '';
  const matches = text.match(regex) || [];
  return [...new Set(matches)];
});
console.log(`✅ ${cnjs.length} CNJs extraídos`);

🔍 Identificar o frame certo

  • Frame raiz: URL = externo_controlador.php
  • Frame de menu: contém "Consulta Processual"
  • Frame de conteúdo: contém CNJs e tabelas
  • Usar frame.url() para debug

⚙️ Alternativa: frameLocator

  • page.frameLocator('#ifrContent')
  • page.frameLocator('iframe[name=conteudo]')
  • Inspecionar o HTML do shell para achar o id correto
  • Pode variar por tribunal

6. Paginação

A lista de processos é paginada. Procurar o link "Próxima" dentro do frame de conteúdo (não no page raiz).

async function goToNextPage(contentFrame) {
  const selectors = [
    'a:has-text("Próxima")',
    'a:has-text("Próximo")',
    'a[title="Próxima"]',
    'a.proximapagina',
    'a:has-text("›")',
    'a:has-text(">>")',
  ];
  for (const sel of selectors) {
    try {
      const btn = contentFrame.locator(sel).first();
      if (await btn.isVisible({ timeout: 800 })) {
        const href = await btn.getAttribute('href').catch(() => null);
        if (href) {
          const url = href.startsWith('http') ? href
            : new URL(href, contentFrame.url()).href;
          // Navegar via evaluate no frame
          await contentFrame.evaluate((u) => { window.location.href = u; }, url);
        } else {
          await btn.click();
        }
        await contentFrame.waitForLoadState('domcontentloaded', { timeout: 20000 });
        return true;
      }
    } catch {}
  }
  return false; // Última página
}

8. Erros comuns e soluções

SintomaCausaSolução
0 processos na lista Extração do documento raiz (não do frame) Usar getContentFrame() e extrair do frame certo
Redireciona para externo_controlador.php Sessão expirada ou URL inválida Verificar se login ainda está ativo; refazer login se necessário
Página em branco no RS116571 Iframe não renderizou ainda Adicionar waitForLoadState('networkidle') + 1500ms de sleep
SC074025 volta para seleção de perfil Dois perfis: RS116571 e SC074025 Detectar estado PROFILE e clicar no botão que contém "SC074025"
Senha não é preenchida no SSO #password pode estar hidden no Keycloak Usar [name=password] com waitForSelector antes de fill
TOTP inválido Relógio fora de sincronia Verificar que o Mac está com horário sincronizado; TOTP tem janela de 30s
page.goto() mata a sessão EPROC não mantém sessão após redirect externo Sempre usar page.evaluate(() => window.location.href = url)
Formulário "Consultar Processos" sem dados Navegou para busca manual ao invés da lista direta Usar a URL com acao=relatorio_processo_procurador_listar

9. Checklist de diagnóstico

  1. Login chegou no PANEL?

    Verificar: page.url() deve conter controlador.php ou painel. Se ainda está no SSO, o login não completou.

  2. SC074025 passou pela seleção de perfil?

    Checar se page.url() contém entrar_sso ou perfil após o login. Se sim, o estado PROFILE não foi tratado.

  3. Quantos frames existem na página? 
    console.log(page.frames().map(f => f.url()).join('\n'));

    Se só aparece 1 frame, o iframe ainda não carregou. Aguardar networkidle.

  4. O frame de conteúdo tem tabelas? 
    for (const f of page.frames()) {
  const tables = await f.evaluate(() =>
    document.querySelectorAll('table').length
  ).catch(() => 0);
  if (tables > 0) console.log(`Frame ${f.url()}: ${tables} tabelas`);
}
  5. Screenshot para debug visual 
    await page.screenshot({ path: '/tmp/debug-eproc.png', fullPage: true });
// Abre a imagem para ver o estado visual da página
  6. Imprimir todo texto dos frames para verificar conteúdo 
    for (const f of page.frames()) {
  const txt = (await f.evaluate(() =>
    document.body?.innerText?.trim() || ''
  ).catch(() => '')).substring(0, 200);
  if (txt) console.log(`[${f.url().split('/').pop()}] ${txt}`);
}
