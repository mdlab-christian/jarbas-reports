Bem-vindo ao MdOs

O assistente de IA da equipe MP Advocacia. Acesse processos, gere petições, consulte prazos e converse com a MIA — tudo pelo terminal, em linguagem natural.

⚡ 19 grupos de skills ✅ 94+ comandos 🦊 MIA ativa 24/7 🏗️ CRUD completo 🆕 Atualização obrigatória
🚀
Como abrir e usar

Três passos. O MdOs já está instalado no seu MacBook.

1

Abrir o MdOs no terminal

cd ~/Documents/MdOs-vault && claude # ↑ cole isso no terminal e pressione Enter
2

Confirmar que está logado

/status # Deve mostrar: "Logged in as [seu@email.com]" # Se não estiver: /login → abre o navegador → faça login com Claude.ai
⚠️

Nunca use API Key

O MdOs funciona com conta Claude.ai. Se alguém configurou ANTHROPIC_API_KEY no terminal, avise o Christian — isso desativa recursos importantes, incluindo o input por voz.

3

Verificar saúde do sistema

/mdos:status # → OlivIA ✅ | Cecilia ✅ | n8n ✅ | Mac Mini ✅ # Tudo verde = pode usar. Algo vermelho = avise o Christian.
🎤

Dica: input por voz

Com conta Claude.ai, segure a barra de espaço para falar em vez de digitar. Funciona muito bem para consultas rápidas como "quais prazos de hoje?" ou "cria atividade para o processo X".

Rotina diária na prática

Como integrar o MdOs ao seu dia a dia de trabalho.

🌅 Início do dia (8h–9h)
# 1. A MIA já enviou o briefing no Telegram às 8h # Leia no celular antes de chegar ao escritório # 2. Abrir o MdOs no computador cd ~/Documents/MdOs-vault && claude # 3. Confirmar prazos urgentes do dia /mdos-controller:prazos # 4. Ver suas atividades pendentes /mdos-atividades:listar → "minhas atividades para hoje"
📝 Gerar uma petição (passo a passo)
# Passo 1 — Verificar jurimetria do juiz antes /mdos-jurimetria:juiz → "perfil do juiz Fulano da Vara X" # Passo 2 — Gerar (demora 8–15 min, pipeline no Mac Mini) /mdos-olivia:peticao → O MdOs vai perguntar: processo? tipo de petição? advogado? # Passo 3 — Revisar antes de assinar /mdos-olivia:revisar → Crítica: coerência, jurisprudências, documentos referenciados # Passo 4 — Ajustar se necessário /mdos-olivia:vibelaw → "reescreve o tópico 3 mais técnico"
🏛️ Registrar sentença e criar execução
# Saiu sentença procedente? Registrar: /mdos-decisoes:criar → "registra sentença procedente no processo 5000123 hoje" # Se procedente, criar execução (cumprimento de sentença): /mdos-decisoes:registrar-execucao → Cria automaticamente o registro de cumprimento

Problemas comuns

Siga estas etapas antes de chamar o Christian.

1

Verificar o status do sistema

/mdos:status # Tudo verde = pode usar. Algo vermelho = avise o Christian.
2

OlivIA sem resposta (mais de 15 min)

/mdos-cecilia:status # → Verifica se o Mac Mini está acessível

O pipeline demora até 15 min. Se passou disso sem resposta, o Mac Mini pode estar com problemas — avise o Christian.

3

Erro de "comando não encontrado" ou ferramenta desatualizada

bash ~/Documents/MdOs-vault/admin/update-tools.sh # Atualiza as ferramentas do MdOs (rodar uma vez depois de cada update)
4

Nada funcionou

/mdos:feedback # → Descreva o problema — Christian recebe notificação

Ou mensagem direta no WhatsApp/Telegram. Não tente consertar o sistema por conta própria.

🔑

Regra de ouro: o MdOs nunca inventa dados

Se a resposta disser "não encontrado" ou "sem resultados", acredite — o dado realmente não está no sistema. Avise o Christian para verificar o cadastro, não tente contornar.
