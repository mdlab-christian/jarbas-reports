🚨 Ações Necessárias
P0 — CRÍTICO

Renovar META_ACCESS_TOKEN

Sem isso, nenhuma mensagem real chega nem sai pelo WhatsApp.

  • Acesse developers.facebook.com → seu app → WhatsApp → API Setup
  • Gere um novo token de acesso permanente (System User token)
  • Atualize META_ACCESS_TOKEN no ~/jarbas/.env
  • Atualize também nas variáveis de ambiente do Railway (n8n)
  • Após renovar, verificar se o webhook da Meta aponta para o workflow correto no n8n
P1 — Alto

Criar workflows: assumir-conversa + devolver-julia

2 botões do MdZap chamam webhooks que não existem no n8n:

  • mdzap/assumir-conversa — deve: UPDATE whatsapp_conversas SET assigned_user_id + enviar timeline event
  • mdzap/devolver-julia — deve: UPDATE whatsapp_conversas SET modo='julia' + limpar assigned
Podem ser criados como workflows simples (~2 HTTP Request nodes cada).
