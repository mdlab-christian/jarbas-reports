Tabelas Criadas
📋 mdzap_julia_prompts

Armazena os prompts da Julia por tipo (sdr_principal, qualificacao, etc) com versionamento. Colunas: id, org_id, tipo, conteudo, versao, ativo, created_at.

SELECT * FROM mdzap_julia_prompts
WHERE org_id = :org_id AND ativo = true
ORDER BY tipo, versao DESC;
📈 mdzap_conversas_timeline

Timeline de eventos de cada conversa: assumiu, devolveu, encerrou, handoff, follow_up. Para o histórico visual no Lead Panel.

SELECT * FROM mdzap_conversas_timeline
WHERE conversa_id = :conversa_id
ORDER BY created_at ASC;
🧠 mdzap_lead_memory

Memória estruturada do lead: preferências, histórico relevante, contexto. JSON indexado por chave. Alimentado pela Julia automaticamente.

SELECT chave, valor, updated_at
FROM mdzap_lead_memory
WHERE lead_id = :lead_id;
📱 whatsapp_contatos

Cache de contatos WhatsApp: número, nome display, foto_url, wa_id, status (ativo/bloqueado/inválido). Sincronizado pelo MDZAP-LEAD-SYNC.

SELECT * FROM whatsapp_contatos
WHERE numero_id = :numero_id
  AND status = 'ativo';
🎵 mdzap_audio_messages

Metadados de mensagens de áudio: storage_path, transcricao, duracao_seg, mime_type, status_transcricao. Alimentado pelo MDZAP-AUDIO-TRANSCRIBE.

SELECT m.*, a.transcricao
FROM whatsapp_mensagens m
LEFT JOIN mdzap_audio_messages a ON a.mensagem_id = m.id
WHERE m.conversa_id = :conversa_id
  AND m.tipo = 'audio';
🔄 fn_mdzap_conversas (RPC)

RPC principal para listagem de conversas. RETURNS TABLE com 17 colunas. JOINs completos com leads, usuários, contatos, última mensagem.

SELECT * FROM fn_mdzap_conversas(
  p_org_id  := :org_id,
  p_status  := 'ativo',
  p_limit   := 50,
  p_offset  := 0
);

⚠️ MDZAP-AUDIO-TRANSCRIBE — Como implementar

O workflow está criado mas é stub. Para implementar com Whisper via HTTP Request nativo do n8n:

// n8n HTTP Request node — Whisper multipart
Method: POST
URL: https://api.openai.com/v1/audio/transcriptions
Authentication: Header Auth (Authorization: Bearer {{$env.OPENAI_API_KEY}})
Body: Form Data (multipart)
  - file: Binary data do áudio (campo: data)
  - model: whisper-1
  - language: pt
  - response_format: json

Input esperado: audio_url (Supabase Storage) → download → Whisper → salva em mdzap_audio_messages.transcricao.
