Contexto — O Que é o Advbox

🏛️ Sistema Legado (Advbox)

Sistema de gestão jurídica usado hoje pelo escritório Midas. Concentra tarefas, intimações, prazos e acesso a tribunais (EPROC RS/SC, JusBrasil SP).

  • Lista centralizada de intimações judiciais
  • Criação de tarefas vinculadas a processos
  • Fluxo manual: Advbox → e-proc → Advbox
  • ~116 intimações ativas por dia (SP)
  • Taskscore: métrica de complexidade/pontuação

🎯 Objetivo da Skill

Criar uma skill JARBAS que valida se o MdFlow Controller está replicando corretamente os dados e fluxos do Advbox — sem dependência manual.

  • Comparar intimações Advbox vs MdFlow
  • Validar que tarefas criadas no Advbox existem no MdFlow
  • Detectar divergências de prazos, responsáveis, status
  • Relatório diário de paridade entre sistemas
  • Alertas proativos quando sync falha

Arquitetura da Skill de Validação
🎯
Objetivo central: A skill skill_validar_advbox.mjs compara os dados do Advbox (via API) com o banco do MdFlow (Supabase) e gera relatório de paridade. Detecta gaps, divergências e falhas de sync antes que causem problemas reais para o escritório.

🔌 Endpoints Advbox a Consumir

GET/api/compromissos?status=pendenteLista intimações pendentes
GET/api/tarefas?responsavel={id}Tarefas por advogado
GET/api/processos/{cnj}Dados de um processo
POST/api/tarefasCriar tarefa
PUT/api/compromissos/{id}/fecharFechar intimação
GET/api/atividades?filtro=julgado_procedenteAtividades julgamentos

Problemas Observados no Advbox (para resolver no MdFlow)

❌ Dores Identificadas

  • Navegação manual entre Advbox e e-proc (cópia de número do processo)
  • Muitas abas abertas simultâneas — queda de produtividade
  • Intimações duplicadas aparecem na lista sem deduplicação
  • Cálculo de prazo manual (15 dias úteis contados na mão)
  • Sem sugestão automática de ação — advogado decide tudo
  • Perda de filtros ao mudar de aba (sessão não persiste)
  • Não alerta sobre prazos críticos pró-ativamente

✅ Como o MdFlow resolve

  • Integração direta EPROC/JusBrasil — sem trocar de aba
  • Interface unificada — tudo em uma tela
  • Deduplicação automática por hash de conteúdo + CNJ
  • Cálculo automático de prazos com calendário de feriados
  • IA sugere tipo de tarefa baseada no conteúdo do evento
  • Estado de filtros persistido por usuário no banco
  • Alertas proativos via Telegram antes do vencimento

Fase 4 — Sprint 2
Relatório Diário HTML + Alertas Telegram

Fase 5 — Sprint 3
Auto-Sync — Preencher Gaps Automaticamente
Para gaps simples (intimação no Advbox mas faltando no MdFlow): importar automaticamente. Para divergências: registrar no log e notificar para revisão manual.

Extrair parâmetros da sentença via Claude (valor, índice correção, dano moral, honorários, AJG). Calcular valor parcial automaticamente. Criar tarefa CS pré-populada.

skill_validar_advbox.mjs — Estrutura Principal

// skill_validar_advbox.mjs // JARBAS — Skill de Validação Advbox × MdFlow import { fetchIntimacoes, fetchTarefas, fetchProcessos } from './modules/advbox-api.mjs'; import { supabase } from '../../lib/supabase.mjs'; import { detectGaps, detectDivergencias } from './modules/comparator.mjs'; import { generateReport } from './modules/report.mjs'; import { notifyChristian } from '../../lib/telegram.mjs'; export async function run(args) { const estado = args.estado || 'todos'; // rs | sc | sp | todos const modo = args.modo || 'full'; // full | incremental | report-only // 1. Buscar dados do Advbox const advboxData = await Promise.all([ fetchIntimacoes({ estado, status: 'pendente' }), fetchTarefas({ estado }), fetchProcessos({ estado }), ]); // 2. Buscar dados do MdFlow (Supabase) const mdflowData = await fetchMdFlowData(estado); // 3. Comparar e detectar gaps/divergências const gaps = detectGaps(advboxData, mdflowData); const divergencias = detectDivergencias(advboxData, mdflowData); // 4. Gerar relatório const report = await generateReport({ gaps, divergencias, estado }); await publishToGitHubPages(report); // 5. Notificar se há gaps críticos if (gaps.criticos.length > 0) { await notifyChristian( `⚠️ Advbox Validation: ${gaps.criticos.length} gaps críticos detectados\n` + `📋 ${report.url}` ); } return { gaps, divergencias, report }; }
