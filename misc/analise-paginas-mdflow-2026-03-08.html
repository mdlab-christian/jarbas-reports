Como usar
Sistema de Ideias — skill page-ideas
Para registrar uma ideia durante uma conversa:
Basta falar: "Registra a ideia de [descrição] na página [rota]"
O JARBAS insere automaticamente em jarbas_page_ideas e confirma.

Ou em qualquer conversa: "anota essa ideia para /clientes: [titulo]"
Para ver o backlog:
"Mostra as ideias pendentes de /processos" ou "backlog de ideias"
Quando executar uma ideia:
Após lapidação ou implementação, o JARBAS marca automaticamente as ideias executadas como status: executada + timestamp. Ideias rejeitadas ficam com status: rejeitada + motivo.
