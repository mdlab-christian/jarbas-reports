01 — Instalar agora
Impacto imediato para o dia a dia
Cada uma resolve um problema real que o JARBAS enfrenta hoje.
📅
⚡ Instalar agora
macOS Calendar
macos-calendar
Lê e cria eventos no Apple Calendar nativamente. JARBAS pode verificar agenda da Eloísa e do Christian, avisar de audiências e criar lembretes de prazos direto no calendário do Mac.
clawhub install macos-calendar
P0
📧
⚡ Instalar agora
Email Daily Summary
email-daily-summary
Loga em contas de email (Gmail, Outlook) e gera resumo diário. Perfeito para o briefing matinal do JARBAS — triagem de emails jurídicos, intimações via email, notificações de sistemas.
clawhub install email-daily-summary
P0
📄
⚡ Instalar agora
PDF Extract
pdf-extract
Extrai texto de PDFs locais com alta fidelidade. Fundamental para o Cérebro Documental — petições, sentenças e acórdãos baixados do EPROC precisam de extração confiável antes da análise IA.
clawhub install pdf-extract
P0
📝
⚡ Instalar agora
Git Essentials
git-essentials
Padrões e comandos git documentados para o agente. O JARBAS já usa git intensamente (commits, branches de lapidação, pushes). Ter a skill garante consistência nos workflows.
clawhub install git-essentials
P1

⚡ Instalar agora
Security Monitor
security-monitor
Auditoria de segurança do host: portas abertas, serviços expostos, atualizações pendentes. Com 5.800 processos e dados sensíveis de clientes, o Mac Mini precisa de health check periódico.
clawhub install security-monitor
P1

KB Collector
kb-collector
Salva YouTube, URLs e textos em base de conhecimento estruturada. Ideal para o JARBAS absorver vídeos de treinamento do João, artigos jurídicos e documentação técnica automaticamente.
clawhub install kb-collector
P1

MiniMax Voice
mm-easy-voice
TTS via API MiniMax. Alternativa ao ElevenLabs para respostas em áudio, mais barata. Pode ser usada para o modo voz do JARBAS enquanto o ElevenLabs não está configurado.

Git Summary
