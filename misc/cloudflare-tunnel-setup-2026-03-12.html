Cloudflare Named Tunnel — Setup JARBAS

URL fixa permanente para o Mac Mini M4. Nunca mais atualizar o .env quando o tunnel cair.

Cloudflare Zero Trust URL Permanente ~20 min
⚠️
Pré-requisito: Precisa de um domínio gerenciado pelo Cloudflare (ex: mdlab.com.br). Se o domínio ainda não está no Cloudflare, transfira o DNS primeiro — é grátis.

Parte 1 — No painel do Cloudflare (browser)

1
Criar conta no Zero Trust 🌐 dash.cloudflare.com

No painel principal do Cloudflare, clique no menu lateral em Zero Trust. Se for a primeira vez, ele vai criar um "team name" — pode colocar mdlab ou qualquer slug.

✅
Plano Free suporta até 50 usuários e tunnels ilimitados. Não precisa pagar nada.
2
Criar o Tunnel 🌐 Zero Trust → Networks → Tunnels
  • 1 Clique em Create a tunnel
  • 2 Escolha Cloudflared (não WARP Connector)
  • 3 Nome do tunnel: jarbas-mac-mini
  • 4 Clique em Save tunnel
  • 5 Na tela seguinte, o Cloudflare vai mostrar um comando de instalação — copie o token (a parte depois de --token)
💡
O token tem este formato: eyJhIjoiXX...LONGA_STRING...XX — salve em algum lugar seguro.
3
Configurar o hostname público 🌐 Zero Trust → Networks → Tunnels → seu tunnel → Edit

Na aba Public Hostname, clique em Add a public hostname:

  • 1 Subdomain: jarbas
  • 2 Domain: selecione seu domínio (ex: mdlab.com.br)
  • 3 Type: HTTP
  • 4 URL: localhost:3012
  • 5 Clique em Save hostname
✅
Resultado: https://jarbas.mdlab.com.br → Mac Mini porta 3012. O Cloudflare cria o registro DNS automaticamente.

Parte 2 — No Mac Mini (Terminal / JARBAS)

4
Autenticar o cloudflared com o token 💻 Terminal no Mac Mini

Execute no terminal (substitua o token pelo que copiou no passo 2):

Terminal
cloudflared service install eyJhIjoiXX...SEU_TOKEN_AQUI...XX
💡
Este comando instala o tunnel como serviço do macOS (launchd) — vai subir automaticamente no boot, mesmo sem login.
5
Iniciar o serviço 💻 Terminal no Mac Mini
Terminal
sudo launchctl start com.cloudflare.cloudflared

Verificar se está rodando:

Terminal — deve mostrar "healthy"
cloudflared tunnel info jarbas-mac-mini
6
Atualizar o .env do MdFlow 💻 Terminal — ou manda "jarbas, atualiza o tunnel" no Telegram

Atualizar o arquivo ~/beta-mdflow/.env.local:

.env.local
VITE_JARBAS_WEBHOOK_URL="https://jarbas.mdlab.com.br"

E fazer rebuild do frontend:

Terminal
cd ~/beta-mdflow
git add .env.local
git commit -m "fix: URL permanente do JARBAS tunnel"
git push
7
Testar o endpoint 💻 Terminal
Terminal — deve retornar JSON com ok: true
curl https://jarbas.mdlab.com.br/health
Resposta esperada
{"ok": true, "service": "jarbas-webhook", "ts": 1773...
✅ Resultado Final
https://jarbas.mdlab.com.br

URL permanente · Sobe automaticamente no boot · TLS grátis pelo Cloudflare · Zero configuração de firewall
