🔧 Como Resolver — Passo a Passo

Etapa 1 — Autorizar chave SSH do Mac Mini no MacBook Air

1
No MacBook Air (você físico, ou Gustavo), abrir terminal e rodar: 
mkdir -p ~/.ssh
echo "ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAILhAOKtjcnRfPA4V6VxjrQBRusfSOmWSe/BpJMDlSzZv jarbas@macmini" >> ~/.ssh/authorized_keys
chmod 600 ~/.ssh/authorized_keys
2
Verificar usuário correto no MacBook Air O usuário SSH deve ser o mesmo do macOS. Rodar whoami no terminal do MBA e me dizer qual é — provavelmente christian ou jarbas
3
Após autorizar, testar com: 
ssh christian@192.168.15.80 "hostname && echo 'JARBAS acessou MBA com sucesso'"

Etapa 2 — Manter MBA acessível sem carregador

1
No MacBook Air, rodar para evitar sleep: 
caffeinate -dis &
Mantém o MBA acordado indefinidamente enquanto tiver bateria. Cancelar com Ctrl+C
2
Configuração permanente (sem precisar rodar todo dia): 
# Adicionar ao ~/.zprofile do MacBook Air:
caffeinate -dis &

Etapa 3 (Opcional) — Instalar OpenClaw no MacBook Air

1
Para JARBAS executar tarefas diretamente no MBA: 
npm install -g openclaw
openclaw start
Depois configuro o Mac Mini para despachar tarefas de dev para o MBA automaticamente via node plugin
