Arquitetura do produto

As camadas
do Jana OS.

Cinco camadas que formam um produto completo — do hardware no escritório até o dashboard no celular do advogado.

Camada 5
🎯 Jana App
Interface do advogado — MdFlow + chat com Jana + dashboard de processos. Web + PWA. O que o cliente toca.
Camada 4
🧠 Jana Brain
O agente: skills jurídicas, memória de processos, modelos de petição, regras do CPC. O que torna Jana inteligente.
Camada 3
⚙️ Jana Core
OpenClaw + runtime de skills + scheduler de tarefas + bridge de delegação. O motor de execução.
Camada 2
🔗 Jana Connect
Integrações: EPROC (RS/SC/SP), WhatsApp, Supabase, n8n, Cloudflare Tunnel. A cola que conecta tudo.
Camada 1
🖥️ Jana Box
Mac Mini M4 dedicado. O hardware físico que roda no escritório 24/7. "A caixa que nunca para."

💡 Por que hardware dedicado é o diferencial

Qualquer SaaS na nuvem tem latência, tem downtime, tem limite de requisições. A Jana Box (Mac Mini) roda localmente no escritório — acessa o EPROC com IP local, processa documentos sem enviar para a nuvem, funciona mesmo com internet lenta. O advogado sente que é "seu" agente, não um software genérico. Isso justifica o preço premium.

Do produto ao negócio. Do técnico ao estratégico. O que torna o Jana OS único e impossível de copiar rápido.

Ideia 01 · Produto

Jana Box — "Plug and play" para escritórios

O advogado recebe um Mac Mini pré-configurado com Jana OS. Liga na tomada, conecta o cabo de rede, aponta o Telegram: pronto. Em 5 minutos o escritório tem um agente autônomo rodando. Nenhum concorrente oferece isso — é produto físico + software + IA numa caixa.

Diferencial máximo
Ideia 02 · Produto

Jana + MdFlow bundled — um produto, dois níveis

MdFlow solo: R$150/mês (SaaS, sem hardware). MdFlow + Jana OS: R$600/mês (inclui Mac Mini alugado ou comprado, Jana rodando 24/7). O advogado que experimenta Jana não volta para o solo. ARR por cliente 4× maior.

Alto ARR
Ideia 03 · Produto

Jana Network — rede de escritórios conectados

Escritórios que usam Jana OS se conectam numa rede. Jurisprudências descobertas por um são automaticamente compartilhadas com todos (anonimizadas). O corpus jurídico cresce com cada cliente novo. Quanto mais clientes, mais inteligente fica Jana para todo mundo — efeito de rede real.

Efeito de rede

Murilo e Camila como revendedores Jana

Canal de vendas
Ideia 05 · Produto

Jana Voice — advogado fala, Jana age

Botão no celular ou no escritório: advogado fala "Jana, cria contestação para o processo 5001234". Jana transcreve, acessa o processo no MdFlow, gera a minuta e manda no WhatsApp. Sem abrir computador, sem digitar. Interface de voz é o próximo frontend.

Futuro próximo
Ideia 06 · Negócio

Jana como "sistema operacional" — não "software"

Posicionamento estratégico: não é "mais um app jurídico". É um sistema operacional — como o Windows era para os anos 90 ou o iPhone para os 2010. O escritório que adota Jana muda como trabalha completamente. Esse posicionamento justifica preço, cria barreira de saída e gera PR espontâneo.

Posicionamento
Ideia 07 · Técnica

Jana Mission Control — dashboard da rede

MacBook do Christian como central de todas as Jana Box. Dashboard em tempo real: saúde de cada nó, tarefas em andamento, alertas, custos. Você gerencia 50 escritórios do mesmo lugar. Detecta quando um nó cai e notifica o parceiro revendedor automaticamente.

Escalabilidade
Ideia 08 · Produto

Jana aprende com o escritório — não é genérica

Cada Jana Box aprende o estilo do escritório: os modelos de petição preferidos, os juízes da comarca, o histórico de vitórias/derrotas, os clientes recorrentes. Após 6 meses, a Jana do Escritório A é radicalmente diferente da do Escritório B — e ambas são melhores que qualquer alternativa genérica.

Moat competitivo
Ideia 09 · Expansão

De jurídico para outros verticais

Jana OS começa no jurídico (porque é onde MdLab tem vantagem). Mas a arquitetura é horizontal. Jana para Contabilidade. Jana para Clínicas. Jana para Imobiliárias. Cada vertical tem um pacote de skills específico. O Core (OpenClaw + Mac Mini) é o mesmo. O mercado endereçável multiplica.

Expansão futura
Ideia 10 · Negócio

Jana como alternativa ao funcionário administrativo

Um escritório de advocacia pequeno tem 1-3 advogados e 1 secretária administrativa. Jana faz o trabalho da secretária: monitora prazos, notifica, organiza documentos, responde clientes (com aprovação), agenda audiências. Custo de R$600/mês vs. R$2.500/mês de salário. O argumento de venda se vende sozinho.
