→ skill_claude_code → OpenClaw → Claude Code com MCP completo (Supabase, GitHub, n8n, Notion). Acesso total ao filesystem do Mac Mini.
/skill [payload]
Executa skill direta (inline ou fila assíncrona). Exemplos: /lapidar_pagina, /captura_intimacoes, /olivia_gerar_peticao
Crons automáticos
EPROC 6h/13h/19h, guardian de prazo 30min, detector de depósito 1h, jurimetria diária 23h, mission control 2min
Telegram → Fila
