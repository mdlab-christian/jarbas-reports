Melhorias de Infraestrutura
Problemas estruturais que afetam estabilidade e escalabilidade
🌐
Named Cloudflare Tunnel
P0 crítico

URL atual muda a cada restart do cloudflared. Isso quebra o n8n toda vez. Named Tunnel = URL permanente jarbas.seutunnel.cfargotunnel.com que nunca muda.


cloudflared tunnel create jarbas
cloudflared tunnel route dns jarbas jarbas.DOMINIO
cloudflared tunnel run jarbas

🧠
OpenClaw Session Manager
P0 crítico

Sessões pesadas (>500KB) com thinking blocks causam crash do gateway. Solução: cron a cada 2h que limpa thinking blocks de todas as sessões ativas, trunca histórico antigo, alerta se sessão >1MB.


Hoje limpei manualmente. Isso precisa ser automático.

📡
JARBAS como Serviço HTTP Permanente
P1

O webhook server na porta 3012 está funcionando, mas não tem supervisão independente. Se o JARBAS cair, o webhook cai junto. Separar o webhook server em processo independente com LaunchAgent próprio.

⚠️
Watchdog mais Inteligente
P1

Watchdog atual detecta sintoma (flood de errors) e reinicia. Precisa detectar causa raiz: sessão corrompida → limpar sessão. Tarefa em loop → cancelar tarefa. Tunnel caído → reiniciar tunnel. Reiniciar JARBAS só como último recurso.

💾
Backup Automático das Sessões
P2

Backup diário das sessões OpenClaw (grupos + DM) para S3/Google Drive. Hoje perdi parte do histórico do grupo OlivIA ao limpar a sessão corrompida. Com backup, dá para restaurar pontos específicos.

🔑
VITE_JARBAS_WEBHOOK_URL no Vercel
P1

Variável de ambiente só está no .env.local (gitignored). Lovable/Vercel não tem → OlivIA em produção não chama o JARBAS. Precisa do Named Tunnel primeiro, depois adicionar no Vercel dashboard.
