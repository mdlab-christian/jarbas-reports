📋 Skill Modo Plano Nova skill

Migrar o modo plano do _planMode Map() em memória para uma skill completa com persistência, histórico e geração nativa de HTML.

💾

Persistência Real

Estado salvo em arquivo JSON — sobrevive a restarts do JARBAS

  • ~/jarbas/data/plan-sessions/[chatId]-[date].json
  • Retoma sessão automaticamente se havia plano aberto
  • Histórico consultável: "qual foi meu último plano?"
🎙️

Áudio Nativo

Transcreve áudios Telegram dentro do modo plano sem depender do OpenClaw

  • Detecta msg.voice e transcreve via Whisper
  • Armazena transcrição com timestamp
  • Acumula igual ao texto
🗺️

Mapa Mental + Plano

Output rico ao encerrar: mapa mental em árvore + plano P0→P1→P2

  • Árvore unicode no chat (sem HTML necessário)
  • Plano de execução ordenado por prioridade
  • HTML opcional com design rico sob demanda
📚

Histórico e Contexto

Sessões anteriores disponíveis para consulta e reutilização

  • /plan lista → últimas 5 sessões
  • /plan retoma → reabre última sessão
  • Injetar plano anterior como contexto em nova sessão

⌨️ Comandos do Modo CC
Comando O que faz
prompt cc: [tarefa] Gera super-prompt com contexto máximo para rodar no CC
prompt cc: [tarefa] — arquivo: [path] Inclui arquivo específico no contexto
cc resposta: [output] Envia resposta do CC para JARBAS atualizar estado da missão
cc status Mostra estado atual da missão orquestrada
cc continua Gera próximo prompt baseado no checkpoint atual
cc inicia [missão] Cria nova sessão de missão orquestrada
