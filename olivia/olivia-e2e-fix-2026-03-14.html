🐛 Bugs encontrados e corrigidos
1

Porta 3012 instável — start.mjs matava webhook-main.mjs

O JARBAS principal (start.mjs) rodava kill -9 $(lsof -ti:3012) a cada restart via heartbeat SIGTERM. Isso matava o webhook-main.mjs (servidor dedicado), criando um gap onde n8n recebia ECONNREFUSED. Foram 30+ registros de "Porta 3012 ainda em uso" nos logs.

✅ Fix: webhook-server.mjs agora detecta se webhook-main.mjs está na porta e pula o kill. Commit: fix: webhook-server não mata webhook-main.mjs no restart
2

N8N sem header de autenticação no nó DOCX Converter

O nó HTTP: DOCX Converter no workflow WH-OLIVIA-GERAR-DOC-V3 não enviava o header mdflow: MdL1501@ exigido pelo webhook-server. Resultado: HTTP 401 Unauthorized quando o servidor estava ativo.

✅ Fix: Header mdflow: MdL1501@ adicionado ao nó via PUT na API do N8N. Workflow atualizado em 11:15 UTC.
3

FK violation — olivia_historico.modelo_id ≠ olivia_modelos_peticao

O nó Salvar Histórico enviava o ID de olivia_modelos_peticao como modelo_id, mas a FK em olivia_historico aponta para olivia_modelos_grupo. INSERT silenciosamente retornava null, deixando historico_id nulo e quebrando o fluxo.

✅ Fix: modelo_id só incluído no INSERT quando data.modelo_grupo_id (grupo válido) está presente. Log de erro adicionado para falhas futuras.
4

N8N (Railway) não consegue acessar localhost:3012 (Mac Mini)

Root cause principal do HTTP 500. N8N roda no Railway (cloud) — localhost resolve para o servidor Railway, não para o Mac Mini. Toda chamada a http://localhost:3012 resultava em ECONNREFUSED ::1:3012.

✅ Fix: Tunnel Cloudflare permanente configurado em jarbas.jarbas-mdlab.com (ID: 4ae4db5d). N8N atualizado para usar a URL pública. URL não muda mais a cada restart.

Regra #1 — Nunca usar localhost em nós HTTP do N8N

N8N roda no Railway. localhost = servidor Railway, não Mac Mini.

✅ https://jarbas.jarbas-mdlab.com/rota   ❌ http://localhost:3012/rota
🔑

Regra #2 — Header mdflow obrigatório em todo nó que chama o webhook

Header: mdflow: MdL1501@ (env: N8N_WEBHOOK_SECRET)
🗄️

Regra #3 — olivia_historico.modelo_id → FK para olivia_modelos_grupo

Não usar ID de olivia_modelos_peticao como modelo_id no INSERT.

if (modelo_grupo_id) histPayload.modelo_id = modelo_grupo_id;
🔌

Regra #4 — Porta 3012 exclusiva do webhook-main.mjs

webhook-server.mjs detecta webhook-main.mjs na porta e pula o kill. Não criar serviços concorrentes na 3012.
