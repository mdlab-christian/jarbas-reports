Gerador de Petições — Arquitetura Atual
📝
skill_olivia_gerar_peticao.mjs Motor Principal
Gera petição seção por seção com streaming. Anti-fingerprint por seed (hash CNJ) em 4 camadas (conectivos, tratamentos, aberturas). Auto-regenera se score < 7 (máx 2x via revisor). Modelos: Sonnet (padrão) + Haiku (rápido). Detecta UF pelo CNJ (8.21=RS, 8.24=SC, 8.26=SP).
🤖
MODO_AGENTE Score 9.7 ✅
Gera petição completa via IA generativa. Puxa contexto estruturado (processo + cliente + empresa + modelo + jurisprudências + playbook). ~90 seg por peça. Resultado mais rico e personalizado.
⚡
EXPRESS Score 7.0 ❌ — Bloqueador
Modo rápido para peças secundárias (Réplica, Agravo, Contrarrazões). Prompt mais enxuto, menos contexto. Problema atual: score insuficiente para Réplicas e peças de recurso. Precisa de refinamento de prompt antes do release.
🧩
SMART_BLOCKS ✅ Estável
Monta petição a partir de blocos de texto pré-aprovados via tags. Zero tokens IA (usa cache de blocos). Muito rápido (<1 seg). Usa até 10 blocos por peça. Ideal para casos padronizados.
📝
VIBELAW ✅ Google Docs OK
Exporta petição diretamente para Google Drive como Google Doc. Integração via OAuth JARBAS (skill_olivia_vibelaw_criar_doc.mjs). ~4 seg. Preview no frontend com layout Midas.
🖨️
skill_olivia_gerar_pdf.mjs
HTML → PDF via Playwright. Times New Roman 12pt, margens ABNT (3cm top, 2cm right). Upload automático para Supabase Storage bucket peticoes. Pronto para protocolo EPROC.
⚖️
skill_olivia_eproc_protocolar.mjs
Protocola PDF no EPROC via Playwright. Suporte: TJRS + TJSC (Phase 1). Fluxo: login → navegar por CNJ → Peticionamento → upload → confirmar → extrair número. Regra P0-04: NUNCA protocola sem confirmação explícita do advogado.
🔍
skill_olivia_gerar_triagem.mjs
Responde perguntas de triagem automaticamente com base nos documentos do processo. Modelo: Haiku (~R$0.02/triagem). Usa ai_agents_config para sistema prompt configurável.
📊
skill_olivia_analista_sentenca.mjs
Analisa sentença → gera playbook de recurso com probabilidade por argumento. Motor: Sonnet. Entrada: texto ou HTML da sentença. Saída: playbook estruturado com estratégia de recurso.
⏰
skill_olivia_prazo_guardian.mjs
Monitora intimações atrasadas. Detecta urgência por keywords (EXTINÇÃO, CANCELAMENTO, SOB PENA, PRAZO FATAL, MULTA...). Cria atividades e notifica via Telegram. Processa até 20 intimações por lote.
