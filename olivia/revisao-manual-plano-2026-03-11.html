Comando de Gravação

Uma linha. Grava tela + microfone simultâneos em MP4:

# Grava tela 0 + fifine mic · 30fps · H264 · qualidade alta
ffmpeg -f avfoundation -framerate 30 -i "0:0" \
-vcodec libx264 -preset ultrafast -crf 28 \
-acodec aac -ar 44100 \
/tmp/revisao_olivia_$(date +%Y%m%d_%H%M%S).mp4
