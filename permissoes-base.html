📋 Como usar este documento

1. Defina a Base Jurídico: liste as rotas que todos do setor jurídico devem ter.
2. Defina a Base Comercial: liste as rotas que todos do setor comercial devem ter.
3. Extras por usuário: para cada pessoa, informe quais páginas adicionais além da base.
4. Mande para o JARBAS e ele aplica tudo direto no banco.

Tags roxas (/pagina) = acesso atual além da base do perfil.
