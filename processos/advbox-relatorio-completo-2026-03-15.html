⚠️ Documentos das pastas Advbox — não acessíveis via API
Testei os endpoints /lawsuits/{id}/attachments, /lawsuits/{id}/documents e /folders — todos retornam 403 Forbidden. A API pública do Advbox não expõe os documentos uploadados nas pastas de processo. O campo folder foi salvo (número da pasta), mas os arquivos em si precisariam de acesso via browser autenticado ou exportação manual pelo painel do Advbox. Alternativa: pedir ao Advbox um export de documentos via suporte, ou acessar via browser automation com Playwright.
