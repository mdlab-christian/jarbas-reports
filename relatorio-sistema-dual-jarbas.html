⚠️ Limitação real identificada: bots Telegram não leem mensagens entre si no grupo
→Telegram bloqueia por padrão que bots recebam mensagens de outros bots (anti-spam nativo)
→Mesmo com allowFrom cruzado, as mensagens de @Jarbas_mdbot não chegam para @Jana_mdbot e vice-versa via polling normal
→Solução real: comunicação via SSH direto (já funciona) ou via Supabase como message bus compartilhado
→Workaround imediato: quando precisar que uma IA "veja" o que a outra fez → pedir explicitamente para ela via ssh mac-mini ou consultar Supabase
✅ Veredicto do JARBAS: sistema funcionando, vale a pena
