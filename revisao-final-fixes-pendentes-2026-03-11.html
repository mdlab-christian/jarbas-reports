Controller — Atalhos de teclado não implementados
RM-05 A4 · Produtividade triagem
P0
Nenhum arquivo do Controller usa useHotkeys ou hotkeys globais. A RM especifica atalhos essenciais para triagem eficiente de intimações:
  • U → marcar urgente | N → marcar normal
  • A → arquivar intimação selecionada
  • → / J → próxima intimação | ← / K → anterior
  • Enter → abrir modal da intimação selecionada
  • Esc → fechar modal
Instalar react-hotkeys-hook (ou usar useEffect com keydown listener) no componente pai do Controller.
// Instalar: npm i react-hotkeys-hook (se não existir) // Adicionar em ControllerIntimacoes.tsx (ou componente pai) import { useHotkeys } from 'react-hotkeys-hook' useHotkeys('u', () => toggleUrgente(selectedId), [selectedId]) useHotkeys('a', () => handleArquivar(selectedId), [selectedId]) useHotkeys('right,j', () => selectNext(), [selectedIndex]) useHotkeys('left,k', () => selectPrev(), [selectedIndex]) useHotkeys('enter', () => openModal(selectedId), [selectedId])
