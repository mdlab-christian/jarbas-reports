
### PASSO 3 — Executar UPDATE para Grupo B
```sql
-- Executar no projeto: qdivfairxhdihaqqypgb
-- Para cada processo do GRUPO B:

SET 
  distribuido = true,
  etapa_distribuicao = 'distribuido',
  data_distribuicao = '{data_do_antigo}'::date,
  distribuicao_manual = true,
  updated_at = NOW()

  AND distribuido = false;
```
